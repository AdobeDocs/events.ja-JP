# Adobe Experience Manager Gems

<img alt="デジタルエクスペリエンス" src="./assets/ADX_Gems.png"/>

**AEM GEM** へようこそ。これは、Adobe AEMのエキスパートによる技術製品の詳細に焦点を当てたAdobe Experience Manager ウェビナーシリーズです。 このシリーズは、製品ドキュメントおよびAdobe Experience Managerに関する技術チャネルを補完するもので、開発者がAEM関連の様々なトピックにアクセスし、詳しく確認することができます。

ウェビナーは定期的に実施されます。  重要なヒント

* すべての AEM GEM は無料です
* 登録が必要です
* セッションに参加できなかった場合は、こちらで録画をご確認ください
* AEM GEM は通常 1 時間で、15 分間の Q&amp;A が行われます
