# Acrobat SignでのWeb フォームの概要

>[!VIDEO](https://video.tv.adobe.com/v/3491645/?learn=on&enablevpops)

フォームをやり取りしたり、契約書への署名を一度に1つずつ追いかけたりしていませんか？ Acrobat Signのweb フォームを使用すると、誰でも署名できるフォームを1つ公開できます。web サイトや共有リンク、埋め込みページなどから、誰でも送信することなくアクセス、入力、署名できます。 アドビのエキスパートによる実践的なウェビナーでは、次の内容を解説しています。

* Web フォームとは？また、その役割とは？人事部では、数百人の従業員が記入できる新規採用のオンボーディングパケットを公開しています。財務チームは、web サイトに顧客向けアプリケーションを投稿して大規模な署名を収集するなど、さまざまな変化に適応しています。
* Web フォームの作成と起動の方法 – ドキュメントまたはテンプレートのアップロード、フォームフィールドの配置、署名者認証の設定、共有可能なURLまたは埋め込み可能なコードとしてのフォームの公開など、最初のweb フォームを作成する方法について、ライブチュートリアルをご覧ください。
* Web フォームの結果を管理および追跡する方法 – リアルタイムで送信を監視し、フォームフィールドデータをダウンロードし、ライブフォームを編集または無効にし、使用状況の増加に合わせてWeb フォームを整理します。

10人または1万人の署名を集める場合でも、このセッションを使用すれば、最初のWeb フォームを自信を持って作成、公開、管理できます。
