# 大規模なデジタルトラストの構築：Acrobat SignによるID検証とセキュリティ

Acrobat SignのID認証、信頼性、コンプライアンスに重点を置いたAdobeの最新イノベーションを常に把握できます。 Acrobat Signの最近のリリースにより、セキュリティ制御の強化と最新のID機能により、あらゆる契約に対する信頼性が向上した様子をご覧ください。 次が含まれます。

* 信頼できるデジタル署名：信頼性、否認防止、長期的な信頼を確保するために、Trust Service Providerの広範なグローバルエコシステムによってサポートされている証明書ベースのデジタル署名。
* グローバルデジタル ID: Digital Identity Gatewayは、オープンバンキングや政府発行のeID スキームなど、世界中で信頼性の高いID認証を実現します。
* 高い保証、規制対応の署名：業界標準およびeIDAS 2.0を含む新しい規制に沿った、高度で適格な電子サイン。
