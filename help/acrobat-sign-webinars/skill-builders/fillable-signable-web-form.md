# 入力可能な署名可能なWeb フォームの投稿

トランザクションを開始したり、署名者の電子メールアドレスを知ったりすることなく、誰でも記入して署名できるweb フォームを作成する方法を説明します。 また、個々の署名トランザクションに使用できるドキュメントテンプレートを作成する方法と、幅広い共有のために一意のURLに投稿する方法についても説明します。

>[!VIDEO](https://video.tv.adobe.com/v/3455444/?captions=jpn&learn=on)

## キーポイント

* Adobe Acrobat Signでweb フォームを作成する前に、アカウントにログインしており、必要な設定が有効になっていることを確認してください。
* Web フォームの設定は、アカウントレベルまたは特定のユーザーグループに対して有効にできます。
* web フォーム設定のオプションには、web フォームの許可、カーボンコピーされた受信者の有効化、PDF プレビュー、署名ブロックでの電子メールのドレスの要求、署名者の電子メール検証などがあります。
* Web フォームに追加の参加者を追加して、ワークフローを作成できます。
* Web フォームを作成するときは、ユーザーグループ、フォーム名、参加者の役割、認証オプションを指定します。
* カウンターサイナー、カーボンコピーされた受信者、自分自身のメールアドレスをワークフローに追加します。
* コンピューターまたはクラウドストレージからドキュメントまたはフォームを添付します。
* フォームフィールドは、手動で配置することも、Adobe Senseiを使用して自動検出することもできます。
* 名前の変更、フィールド割り当ての調整、フィールドタイプの指定、フィールドを必須にすることで、フォームフィールドをカスタマイズできます。
* Adobeでは、ヘルプセンター、Experience League、サポートコミュニティ、スキルビルダーウェビナー、リソースハブなどのリソースを提供しており、ドキュメントプロセスの詳細を確認できます。
