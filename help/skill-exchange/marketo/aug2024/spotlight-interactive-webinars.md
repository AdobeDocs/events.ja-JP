# インタラクティブウェビナーで、バーチャルイベントを効率化しましょう

インタラクティブウェビナーを使用してウェビナーやイベントを強化することで、時間を節約できます。プロモーションメールの作成やスマートリストを使用したセグメンテーション、ライブウェビナーの開催、パフォーマンス分析へのアクセス、フォローアップの自動化など、ワークフロー全体を1か所で管理できます。

このセッションでは、次のことを説明します。

* 最初のインタラクティブウェビナーの設定と構築に関するベストプラクティス
* レポートで成功を追跡し、投票を活用してリードを生成し、フォローアップを自動化する方法
* インタラクティブウェビナーによって、展開にかかる時間を短縮しながらエンゲージメントを高めることができる方法に関する、実際のユースケースと結果
