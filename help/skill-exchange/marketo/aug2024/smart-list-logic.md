# スマートリストロジックを征服してキャンペーンオーディエンスにリーチ

スマートキャンペーンは、Marketo Engageの自動化機能を推進する強力なエンジンです。重要なコンポーネントは、「スマートリスト」タブを使用して、適切なオーディエンスをターゲットにすることです。 スマートキャンペーンの複雑な機能、フィルターとトリガー、高度なスマートリストロジックを解説します。 マーケティングプログラムの目標を達成するために、スマートキャンペーンの可能性を最大限に活用する方法について解説します。

## 重要な留意点

* スマートキャンペーンの概要と、Marketo Engageのアーキテクチャ全体との連携方法
* オーディエンスの設定とカスタマイズをおこなうための4つのロジックのレベルについて詳しく説明します
* ロジックをトラブルシューティングし、ターゲットとするオーディエンスが正しくターゲティングされていることを確認する方法
