# A/B テストによるパフォーマンスの向上と検証

アドビのA/B テストを活用して、Adobe Marketo Engageのマーケティング戦略を変革しましょう。プログラムを管理されたテストに移行し、測定可能な成果を達成する方法を解説します。 メールやランディングページのバリエーションをテストすることで、オーディエンスの心に最もよく響くものを見つけ出し、その理由を明らかにできます。 このセッションでは、次のことを説明します。

* 仮説の展開：クリックスルー率に対するダイレクトcall-to-actionの影響など、何が起こり、なぜ起こるかを予測します。
* テスト対象の変数：Marketo Engageでテストできる様々な要素（見出し、call-to-actionボタン、画像、コンテンツのレイアウトなど）を検討します。
* セグメンテーション：A/B テストのために効果的にオーディエンスをセグメンテーションする方法を理解して、戦略を洗練させ、より良い結果を達成します。
* 結果の解釈：統計的有意性の計算方法と、実用的なデータを収集するために適切な期間テストを実施することの重要性。
