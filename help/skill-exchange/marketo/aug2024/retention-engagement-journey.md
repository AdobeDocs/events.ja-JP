# 顧客エンゲージメントデータのインサイトを活用した強力なリテンションおよびエンゲージメントジャーニー

複数のブランドや製品ラインを管理していますか？ または、オーケストレーションが必要なメールナーチャリングが多数存在しますか？ このセッションでは、チャネルをまたいで顧客接点をつなぎ合わせ、顧客エンゲージメントデータを活用して、顧客維持とエンゲージメントの段階で詳細にパーソナライズされたコンテンツを提供する方法を説明します。

このセッションでは、次のことを説明します，

* Adobe Marketo Engageで顧客データとレポートを分析し、そのインサイトにもとづいて、オーディエンスを効果的にセグメンテーションしましょう。
* さまざまなプログラムの種類や自動化手法を利用して、オーディエンスをセグメンテーションし、エンゲージメント、製品販売、更新の段階を通じてシームレスに移行できます。
* 個人またはプログラムレベルのトークンと速度スクリプトを利用して、詳細にパーソナライズされたコンテンツを制作できます
