# マルチストリームジャーニーの力を引き出す：エンゲージメントプログラムの高度なアプリケーション

Marketo Engageのマルチストリームのエンゲージメントプログラムを利用して、リードジェネレーションとセールスパイプラインのアプローチを変革する方法を紹介します。 このセッションでは、さまざまなビジネスモデルにわたる実際のエンゲージメントプログラムのユースケースからインスピレーションを得ることができます。 ビジネスモデルにもとづいてカスタマージャーニーのステージを定義し、Adobe Experience Manager Streamsを活用してライフサイクルプログラムを作成し、すぐに利用できるレポート機能によって一般的なライフサイクルレポートのニーズを満たす方法を紹介します。 このセッションでは、次のことを説明します。

* さまざまなビジネスモデルの例を用いて、カスタマージャーニーのステージを定義します。
* カスタマージャーニーの設計図をマルチストリームのエンゲージメントプログラムに変換。
* 顧客ライフサイクルをまたいでエンゲージメントプログラムを分析するために、ネイティブレポート機能を活用します。
