# Marketoのロードマップ

Marketoは、柔軟性と拡張性に優れているので、多くのチャンスを生み出すことができます。 しかし、同時に、プランニングの不足やプランニングの欠如により、Marketoのインスタンスの拡張性が低下し、技術的な負債が大量に発生する可能性があります。

Marketoインスタンスを商品として捉えるのに役立ちます。 顧客もいるため、インスタンスを顧客満足度の高い製品に開発する必要があります。 しかし、ライフサイクルマーケティングやABMに急いで取り組むことは、最終的にはうまくいかないかもしれません。 また、製品ロードマップという、作業プログラムを策定し、それを実現する方法を指導します。
