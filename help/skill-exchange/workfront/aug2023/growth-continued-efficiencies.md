# Adobe Experience Makers スポットライト – 成長に備え、効率性を継続

Workfrontを実装し、最初のワークフローを正常に開始しました。これで、Workfrontが動作し、全員が必要とする単語が表示されました。

「持つのは良い問題です」と言われますが、あなたのインスタンスを拡大することは何を意味しますか？ アドビのExperience Makers スポットライトセッションでは、3つの企業がWorkfrontインスタンスの拡大と拡張にどのように取り組んできたのかを紹介します。

よくわかると思いますが，

* Thermo Fisher Scientificのマーケティングテクノロジスト兼プロジェクト管理Workfront責任者であるCathy Glenn氏が、適切な人材とリソース（グループ管理者の活用を含む）を確保できていると考える理由 成長に向けた重要な最初のステップです
* Amazon Web Services（AWS）のマーケティングオペレーションマネージャーであるTrinite Bryant氏は、Workfrontに対するコミュニティとガバナンスの意識を高め、利用頻度と参加率を向上させることで、ビジネスの成長を促進するのに役立っています
* Deloitteのテクニカルプログラムおよびプロジェクト管理マネージャーであるTim Brooks氏が、拡張を合理化し、継続的な効率性を見出すために、Workfrontを製品化およびテンプレート化した方法をご確認ください
