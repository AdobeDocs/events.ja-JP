# 目的に応じた自動化

自動化を活用することで、ユーザーは時間のかかる日常的な反復作業ではなく、独自のスキルや専門知識が必要な作業に集中できるため、効率性の大きな可能性を引き出すことができます。

このセッションでは、Zimmerman AdvertisingのJennafer Higgs氏、Jonathan Cabre氏、Malcom Benites氏が、より効率的なワークフローを構築できることが証明されている自動化機能の実例を紹介します。

Nickは、次のようなFusionと非Fusionの両方の自動化を共有します。

・ 自動的にプロジェクトを閉じる方法
・ サブテンプレートの生成
・ 統合された承認済みアセットのレビュー
