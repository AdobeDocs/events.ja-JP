# Workfrontのオブジェクトについて

Workfrontでのオブジェクトの理解は、新しいシステム管理者が習得できる最も重要な概念の1つです。 様々なオブジェクトとそれらの間の関係を理解すると、空が限界になります。 ポートフォリオ、プログラム、プロジェクト、タスク、イシュー/リクエスト、企業、グループ、チームなどの人物オブジェクトなどのプロジェクトオブジェクトのニュアンスについて説明します。 オブジェクトの向き101を考えてみましょう。
