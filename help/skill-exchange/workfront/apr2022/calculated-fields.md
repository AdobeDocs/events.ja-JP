# ヒントとテクニック：誰もが知っておくべき計算フィールドとそのレポート方法

プロジェクトの更新ストリームの複数のページをスクロールして、プロジェクトが現在に移行した日付を確認したことがありますか？ これは、容易にアクセスでき、レポートで使用できる計算フィールドでキャプチャできる情報の優れた例です。

このセッションでは、次のことを説明します。

* Workfrontでのレポート作成を容易にする、Anthonyのお気に入りの計算フィールドの計算
* ニーズに合わせて計算を調整する方法
* カスタムフォームの計算フィールドに重点を置く（これらの例の多くは、レポートのカスタム列の計算に使用できますが、カスタムフォームの計算フィールドに重点を置きます（コレクションやEXIST フィルターなどについては説明しません）
