# ユーザーの導入を克服するための考慮事項

Workfrontのような作業管理ソリューションの導入は、さまざまな側面を持ちますが、最も基本的なことのひとつは、組織内の誰が作業管理ツールを使用し、どのようなメリットを享受できるのかを把握することです。

このセッションでは、誰を中心に取り組むのか（例：「who」）を正確に把握し、組織が採用の「なぜ」に関して直面する一般的な課題を特定し、検討します。

主な内容：

* 導入アプローチにおいて割引できない5つの主要ペルソナ
* Workfrontの導入に伴う主な課題は、これまで以上に考慮する必要があります。
* Workfrontのエンドユーザー導入状況の把握に役立つリソースを確認する（または、自己満足したユーザーや離れたユーザーと再エンゲージする）
