# 高度なテキストモードでWorkfront レポートを強化

高度なテキストモード機能により、Workfrontレポートを次のレベルに引き上げることができます。 このセッションでは、AdobeのNathan Johnson氏が、より効率的でダイナミックで使いやすいレポートを作成する方法を紹介します。

* **わかりやすいレポート用のフィールドの結合**&#x200B;複数のデータポイントを1つの列に組み合わせることで、乱雑さが軽減され、読みやすさが向上し、重要な情報がハイライトされます。
* **値式を使用したカスタム計算の使用** リアルタイム計算と条件付きロジックを適用して、カスタム要約、指標、またはステータスメッセージを表示し、基礎データを変更することなくレポートインサイトを強化します。
* **柔軟なフィルタリング用のカスタムプロンプトの作成**&#x200B;動的なフィルターをレポートに追加し、エンドユーザーがレポートを編集せずにビューを調整できるようにし、レポートをよりインタラクティブでユーザーフレンドリーなものにします。
* 洗練された機能的なレポートを作成し、有意義なインサイトを即座に提供する方法をご確認ください。

## データを実用的なストーリーに変換

Storytellingに技術的な専門知識を組み合わせることで、Workfrontレポートの効果を引き出すことができます。

* ** 高度なテキストモード **デフォルトのUI制限を超える、柔軟にカスタマイズされたレポートを有効にします。
* **列を結合**&#x200B;すると、乱雑さが軽減され、読みやすさが向上します。これにより、主要なプロジェクトの詳細を単一のビューで表示できます。
* **値の式** プロジェクトの状態やスケジュールの計算など、リアルタイムのインサイトをレポート内で直接提供します。
* **カスタムプロンプト**&#x200B;多様なチームのニーズに対応し、ユーザーがレポートを動的にフィルタリングおよび操作できるようにします。

これらの手法を活用することで、情報を提供するだけでなく、オーディエンスを導き、エンゲージするレポートを作成することができます。 この知識をもとに、独自の意見を反映したレコードを作成し、より優れた意思決定を促進できます。
