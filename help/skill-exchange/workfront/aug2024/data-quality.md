# 混沌から明瞭化へ：データ品質を向上

「ダーティ」データが原因でWorkfront レポートが不足していますか？ 「お客様だけではありません。 データ品質が低い場合、日々の業務や意思決定プロセスに大きな影響を与える可能性があるため、この重要な問題に対処することは不可欠です。 カスタマーサクセスアーキテクトであり、以前は顧客であったJen Desmond氏と連携して、Workfrontで自社のデータを活用する方法を解説します。

このセッションでは、次のための実用的なヒントと戦略をご紹介します。

* データクリーンアップ計画の策定と実行
* データ品質の問題を明らかにするために使用できるレポートの例
* あらゆるペルソナの利用者によるデータの整理と維持
* ボーナス！ Canvas ダッシュボードの機能を拡張するためのデータの準備」
