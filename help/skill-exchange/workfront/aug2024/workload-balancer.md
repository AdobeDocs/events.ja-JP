# リソースエクセレンス：ワークロードバランサーのカスタマイズ

常に、利用可能なリソースについて把握することに課題がありますか？ 2～3か月後のリソースのキャパシティについて、どのように把握するのか？ これらの課題に精通している場合は、お気に入りください。 このセッションでは、Copper Giants （Liberty Mutual Insuranceの社内広告代理店）のSteven Enos氏が、Adobe Workfront Workload BalancerとResource Plannerを設定して管理し、最大限の効果を得る方法について説明します。 具体的には、以下を共有します。

* あるツールを別のツール（ワークロードバランサーまたはリソースプランナー）よりも選択する場合、または両方を使用できる場合
* フィルターとビューをカスタマイズして、レポートに必要なデータを正確に表示する方法
* 正確なキャパシティプランニングに休暇を入力することが重要な理由
* Steve氏のチームは、Workfrontのリソース管理ツールを活用することで、今後3 ヶ月間の日々のリソースキャパシティと将来のリソースプランニングを決定しました
