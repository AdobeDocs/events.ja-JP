# 2021年に知っておくべきAnalysis Workspaceのヒントとテクニック

Analysis Workspaceは、今日知られている最高の、最も汎用性の高い分析インターフェイスです。」と、Adobe AnalyticsチャンピオンのFrederik Werner氏は述べています。 このセッションでは、彼は彼のお気に入りのヒントを共有するつもりです&amp;あなたのビジネスがより多くのデータ主導になるのを助けるために、あなたの関係者がより多くのセルフサービスになるために、そしてあなたの企業のデータサイエンティストがあなたの手に持っている素晴らしいツールを嫉妬するために。 これには、彼のブログ https://www.fullstackanalyst.io/のベストオフに加えて、ROIを11まで向上させるための共有されていないヒントが含まれます。 さらに、ライブ Q&amp;Aで質問をする機会もあり、これは間違いなく見逃したくないチャンスです。
