# 共通指標の明確化

私たちは皆、毎日使用する多くの指標があります。 このセッションでは、それらが実際に何を意味するのか、そして最も効果的にそれらを使用する方法について説明します。 訪問や訪問者などの指標をすぐに使用できるだけでなく、コンバージョン率や直帰率などの計算指標も利用できます。

>[!VIDEO](https://video.tv.adobe.com/v/3471114/?learn=on&enablevpops)

## 直帰率とシングルページ訪問の比較

* **バウンス**&#x200B;は、1回のヒットのみを含む訪問です（例：ページ読み込みのみ、インタラクションなし）。
* **単一ページ**&#x200B;の訪問は、一意のページが1つだけの訪問ですが、複数のヒット （ポップアップ、Cookie プロンプトなど）が含まれる場合があります。
* Adobeのデフォルトのバウンス率は、最初のページで呼び出しが実行された場合、真のバウンスを下回る可能性があります。
* 単一ページへの訪問を使用して直帰率をカスタマイズすることで、離脱したユーザーの状況をより明確に把握できます。
* ページレベルのバウンス分析では、訪問ではなくエントリを使用して、精度を向上させます。

## Web分析指標の力を引き出す

最も一般的なweb分析指標を理解し、より優れたビジネス上の意思決定に活用する方法をご確認ください。

* **指標の定義** ユニーク訪問者、訪問回数、ページビュー、ページイベント、オカレンスの違いを説明します。
* **バウンス率のインサイト** バウンス率とシングルページ訪問の微妙な違いと、カスタムバウンス率がより正確になる理由を理解します。
* **コンバージョン率の明確性** コンバージョン率、1訪問あたりの注文、注文件数の違い、適切な指標を選択することが重要な理由を確認できます。
* **ボットトラフィック認識** ボット指標を使用して、よりクリーンなレポートのために人間以外のトラフィックを特定し、フィルタリングする方法を説明します。

これらの概念を習得することで、ニーズに最適な指標を選択し、分析レポートの価値を最大化することができます。
