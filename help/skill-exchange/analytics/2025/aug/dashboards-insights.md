# ダッシュボードがインサイトを提供しない理由（実際に何をしているのか）

「インサイトをもたらすダッシュボードが必要です」 お馴染みですか？ ダッシュボードはインサイトを提供するように設計されておらず、出発点にすぎません。

このセッションでは、ダッシュボード、ナラティブレポート、詳細な分析という3つのツールを使用して、数値の背後にある理由を実際に明らかにする実用的なフレームワークを紹介します。 明確なシステムによってデータから行動を生み出し、最終的には空のダッシュボードという無限のサイクルから抜け出しましょう。

>[!VIDEO](https://video.tv.adobe.com/v/3471120/?learn=on&enablevpops)

## データを有意義なアクションに変換

データの力を引き出すには、ダッシュボードだけでは不十分です。そのためには、観察データを実用的なインサイトに変えるシステムが必要です。

* **Insight定義** insightとは、重要なパターンや異常値を明らかにし、コンテクストに適用することで意味のある変化を促す、理解の転換を指します。
* **3層システム** ダッシュボード（双眼鏡）のスポット傾向、ナラティブリポート（拡大鏡）のコンテキストの追加、および深層分析（顕微鏡）が推奨事項を提供します。
* **重要な要素**&#x200B;真のインサイトを得るには、パターン、異常値、コンテキストを交差させる必要があります。
* **実用的な成果**&#x200B;優先順位付けフレームワークと、ビジネスへの影響に関するループを終了することで、インサイトが測定可能な改善につながることを確認します。

このアプローチは、利用者が静的なデータから動的でビジネスを変える意思決定へと移行するのに役立ち、分析を真に価値のあるものにします。

## 真のインサイトの定義

* インサイトは単なるデータポイントではありません。パターン、異常値、適切なコンテキストの両方が必要です。
* インサイトを獲得するための4つの重要なトリガー:
   * **接続**&#x200B;以前は関係がなかったデータ間の関係を明らかにする。
   * **矛盾**&#x200B;期待と現実の間に矛盾が見られます。
   * **一致**&#x200B;予期しない整列または相関関係を特定しています。
   * **好奇心**&#x200B;期待に反する異常を調査しています。
* コンテクストがなければ、最も興味深いデータでさえノイズです。
* 変化を促す実用的なインサイトは、価値を実証するために測定可能でなければなりません。
