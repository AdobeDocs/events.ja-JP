# Skill Exchange Event 2023年8月 – Learn track - Adobe Analytics: Analyzing the data

フリーフォームテーブルは、Analysis Workspaceの柔軟な基盤です。これらの強力なテーブルを使いこなせれば、Workspaceのジャーニーを大きく進めることができます

セグメントは、データをフィルタリングおよび詳細に分析するための非常に柔軟な方法であり、グローバルに、指標レベルで適用することも、スタンドアロンディメンションとして使用することもできます

パネルには、複雑な分析を行うためのショートカットが用意されています。特に、クイックインサイトパネルは、ビジネスに直面する疑問に答えるための優れたオプションです
