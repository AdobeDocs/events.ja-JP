# スキル交換イベント 2023年8月 – Grow Track – 顧客体験を向上させるためのコンバージョン防止要因の分析

どの業界でも、オンサイト体験を継続的に改善することが重要です。 コンバージョン対策の目的は、顧客のオンサイト体験に悪影響を与えている要因を把握し、その解決を目指すことです。 エクスペリエンスを向上させるために確認すべき指標は、数多くあります。

このセッションでは，

* エラー率を活用してインサイトを獲得する方法
* カスタマージャーニーを改善するための顧客の混乱すべき領域の特定
* 組織にコンバージョン防止を導入するためのヒント
