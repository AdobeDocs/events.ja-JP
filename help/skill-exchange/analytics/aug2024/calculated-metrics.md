# 大きな指標には大きな責任が伴います

このセッションでは、計算指標を利用して、組織の分析を次のレベルに引き上げるのに役立つ方法について説明します。 管理者チームと連携して、適切な指標を作成するだけでなく、適切な指標を常に利用できるように、他のアナリストを適切な指標にするだけでなく、それらを整理しておくのに役立つことはたくさんあります。 利用可能なさまざまなオプションを使用して、計算された指標をクリーンで整理するための戦略を作成する方法について説明します。 また、さまざまなプリセット機能を使用して高度な指標を構築するためのヒントも紹介します。

## 重要な留意点

計算指標のカタログを利用できる利点
管理者としての役割を使用して、分析ユーザーが適切な指標を使用していることを確信できるようサポートする方法
高度な指標作成のヒントと、その他のリソースを検索する場所
