# デジタルハーモニーを構築：AEMのカスタマイズとガバナンスで一貫性のある体験を構築

この高度なセッションでは、デザインとエクスペリエンスの構築を強化するために、Adobe Experience Manager（AEM）内で要素をカスタマイズすることの複雑さについて詳しく説明します。 このセッションでは、音楽的なアナロジーを使用して、コンポーネント、テンプレート、コンテンツフラグメント、エクスペリエンスフラグメントのカスタマイズを検討しながら、ブランドとビジネスのガイドラインを遵守します。 このセッションでは、ユーザーの共感を呼び、システムアーキテクチャの成熟度をサポートする、一貫性のあるデジタル体験を構築するための包括的な知識を提供することを目的としています。

## 重要な留意点

* メロディを探す：AEMの要素をどこでどのようにカスタマイズするかを決めます
* ミュージックを調和させる：微妙な違いに対応するためにバリエーションを活用する
* 常に調整：デザインとビジネスガバナンスのためにビルドを活用する
