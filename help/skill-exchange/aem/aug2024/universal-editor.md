# ユニバーサルエディターを備えた次世代のAEM Sites

「Edge Delivery Servicesの最近のイノベーションにより、Adobeをご利用のお客様は、超高速なエクスペリエンスとSEOの向上により、非常にパフォーマンスの高いサイトを構築できるようになりました。しかもどのサイトも、はるかにアクセスしやすいスキルセットを備えています」 ユニバーサルエディターは、AEM Sitesが知るビジュアルなインコンテクスト型オーサリングとコンテンツガバナンスを提供しながら、Edge Delivery Servicesで優れたエクスペリエンスを提供することで、さらに次のステップへと進みます。 Adobeのテクノロジーを活用して、顧客体験を向上させる方法をご紹介します。

## 重要な留意点

* ユニバーサルエディターのカスタマーゼロプロジェクトであるExperience Leagueを率いたプリンシパルコンサルタントの経験を聞きます
* ドキュメントベースのオーサリングやAEM Sites ページエディターと比較した場合の主な違いをデモでご確認ください
* Edge Delivery Servicesのビジネス価値と、ユニバーサルエディターを今すぐ使用してAEM Cloud Service チームを開始する方法について説明します
