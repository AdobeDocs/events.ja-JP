# DAM の成功の基盤：メタデータ、コンテンツ構造および分類

このセッションに参加して、顧客が DAM のパフォーマンスを低下させ、ユーザーの採用率の低下につながる一般的な戦略の間違いに関するインサイトを得ます。 カスタマーサクセスマネージャーの Andrea Hardeman とAEM アーキテクトの Tommy Miller が、複合シナリオベースのアプローチを順を追って説明し、他のお客様の課題から学んだことを活用する方法を示します。

## 重要な留意点

* 砂利と土を使用してメタデータや分類のプラクティスを構築する際に、顧客が直面することが多い課題をより深く理解する必要があります
* メタデータ、コンテンツ構造、分類に対する全体的なアプローチを通じて、強力な基盤を使用してビジネスプラクティスを強化する方法を説明します
