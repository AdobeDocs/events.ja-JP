# DAMの成功の基盤：メタデータ、コンテンツ構造、タクソノミー

このセッションでは、顧客がDAMのパフォーマンスを低下させ、ユーザー導入率の低下につながる一般的な戦略の失敗に関するインサイトを提供します。 カスタマーサクセスマネージャーのAndrea Hardeman氏とAEMのアーキテクトであるTommy Miller氏が、合成シナリオベースのアプローチを通じて、顧客のほかの課題から得た知見を活用する方法を解説します。

## 重要な留意点

* 顧客が直面する課題について、より深く理解したうえで、メタデータと分類の実践を砂利と土壌で構築します
* メタデータ、コンテンツ構造、分類基準に対する包括的なアプローチを通じて、自社のビジネス慣行を強固なものにする方法を学びましょう
