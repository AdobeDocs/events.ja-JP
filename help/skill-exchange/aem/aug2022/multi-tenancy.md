# AEM Champion セッション マルチテナンシー

このセッションでは、同じAEM as a Cloud Service インスタンスに複数のテナントを存在させる方法に焦点を当てます。 Adobe Cloud Managerを使用して、同じAEM インスタンスに複数のコードリポジトリをデプロイすることに重点を置きます。 これには、さまざまなGit リポジトリの設定や、Git サブモジュールの使用が含まれます。
