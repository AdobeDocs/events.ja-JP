# Skill Exchange Event 2023年8月 – Adobe Experience Maker スポットライト - AEMを活用して、詳細にパーソナライズされたマーケティングメールを作成する

メールコミュニケーションは、既存顧客と潜在顧客にリーチするための最も重要な方法のひとつです。 非常にパーソナライズされたメールを効率的に作成できるため、顧客エンゲージメントを大幅に向上させることができます。 AEM（Adobe Experience Manager）を思い浮かべる人は、web サイトやweb コンテンツを思い浮かべることがよくあります。 AEMは、電子メールの作成にも使用できます。

このセッションでは、T. Rowe Price氏が、AEMのメールコアコンポーネントの構築に貢献し、スケーラブルで高度にパーソナライズされたメールテンプレートの構築に活用した方法を解説します。 これらのコアコンポーネントを、Adobe CampaignやAdobe Journey Optimizerなどのメールオーケストレーションサービスと統合するために利用できる様々なオプションを検討します。 また、コンテンツフラグメントとエクスペリエンスフラグメントを活用して、マルチチャネルコンテンツを制作する方法についても解説します。 最後に、メールテンプレートをパーソナライズし、セグメント化するための新しく考案された方法を紹介します。
