# Skill Exchange Event 2023年8月 – AEM Web Track - Experience Makers スポットライト

T.Rowe PriceのGreg Dimeris氏が、コンテンツの構成方法やスタック方法、スーパーチャージ方法を解説します。 スピードと成長を実現するための構成要素としてコンポーネントを使用して、Web エクスペリエンスを強化する方法について説明します。

LPL FinancialのCindy Underwood氏が、Adobe Experience Manager（AEM） Adobe Experience Manager Sitesの価値を引き出すために、Adobe Bridgeが最適なツールである理由を解説します。 Adobe Bridgeなら、アセットの命名規則やサイズ変更、フォーマット、カラープロファイルの更新を数秒ですばやく実行できます。 AEM SitesのステージングプラットフォームとしてAdobe Bridgeのメリットを引き出す方法について説明します。
