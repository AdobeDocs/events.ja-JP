# バリューマッピングと価値実現のためのロードマップ

適切なテクノロジーを身につけることは、出発点にすぎません。企業は、どのように投資の価値を最大化できるのでしょうか。 このセッションでは、企業がビジネス価値に関連して、テクノロジーのユースケースをどのように開発、評価、優先順位付けできるかを説明します。 続いて、定量化可能な成果と優れた顧客体験を促進するために、そのような価値を正確に把握できるKPIを設定し、レポートを作成する方法について解説します。
