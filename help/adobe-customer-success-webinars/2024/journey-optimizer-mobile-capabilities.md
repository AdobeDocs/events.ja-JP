# Journey Optimizer Mobile Capabilities

このAdobe Journey Optimizer入門セッションでは、初心者がパーソナライゼーションを活用して成果を上げるSMS/プッシュチャネル CXを作成できるよう支援します。 フラグメントの使用とパーソナライゼーションについてお話します。

>[!VIDEO](https://video.tv.adobe.com/v/3433001/?learn=on)

## 重要ポイント上位5つ

**Agile Essentials モバイル機能**

* Journey Optimizerを利用して、モバイル機能を最適化する。
* さまざまなチャネルをまたいでパーソナライズされたメッセージを配信し、顧客データを統合してパーソナライズされた体験を実現できます。

**SMS設定の設定**

* サードパーティのSMS サービスプロバイダーとの統合とAPI資格情報の設定の重要性。
* 顧客との効果的なSMS コミュニケーションのためのチャネルサービスの構築。

**プッシュ通知設定**

* パーソナライズされたコンテンツとメディアによるオーディエンスの定義、スケジューリング、プッシュ通知の作成。
* Fireflyのような生成AIを利用して、ブランドのスタイルに合わせてパーソナライズされた画像を作成。

**アプリ開発とサービス**

* 正しい管理コンソール、プロパティ、会社権限を使用したアプリサービスの設定。
* ユーザーエンゲージメントのためのアプリ内通知の設定と、オーディエンスセグメントにもとづくメッセージのパーソナライズ。

**テンプレートの複製と再利用**

* は、テンプレートとフラグメントを複製および再利用して、時間を節約し、キャンペーン全体でメッセージとブランディングの一貫性を維持する効率を強調しました。
* 一度制作したコンテンツを複数のキャンペーンに複製し、今後のマーケティング施策で効率化できるメリットを強調しました。
