# Adobe Real-time CDP - ソリューション機能のユースケースマッピング

このウェビナーでは、ユースケースを調査および優先順位付けし、それらを特定のReal-Time CDP機能にマッピングし、ビジネス成果と価値との整合性を確保するための主要な方法を解説します。

>[!VIDEO](https://video.tv.adobe.com/v/3429290/?learn=on)

## 主な討議ポイント

* Adobe Real-Time CDPは、ビジネス成果に特化したユースケースを解決し、優先順位を付けます
* 優先度の高いユースケースに合わせたAdobe Real-Time CDP機能の調整

## 会合の概要

**Real-time CDPが解決する課題**

* 個別のマーケティングツール
* 散在データの生成元
* ガバナンス規制の変化

**サードパーティ Cookieの廃止の影響**

* 有料メディアのリターゲティングやパーソナライゼーションなどのユースケースへの影響

**Adobe Real-Time CDPとDMP Audience Managerの比較**

* データ収集、オーディエンス管理、データ出力構造に重点を置く

**整合と優先順位付けの重要性**

* CDP能力を戦略的な優先順位に整合させる
* ユースケースを優先する
* 確実に実行

**件の戦略について話し合いました**

* Adobe Marketo Engageの高度なセグメンテーション機能の活用
* データ共有機能の活用
* パーソナライズされた顧客体験と維持戦略へのCustomer Airの活用
