# AEMの基本AEM オーサリングの概要

AEMが、従来のweb ページからヘッドレスCMSの設定に至るまで、さまざまなコンテンツ制作手法をサポートする、堅牢かつ柔軟なオーサリング環境を提供する方法をご確認ください。

## 主な討議ポイント

* ユニバーサルエディターを含むオーサリング
* コンテンツフラグメント
* コラボレーション機能

>[!VIDEO](https://video.tv.adobe.com/v/3435747/?learn=on)

## 重要な留意点

### AEM オーサリングの基本

このセッションでは、主要な概念や機能を含む、Adobe Experience Manager（AEM）オーサリングの基本について説明しました。

### AEM オーサリングモデル

AEMのデュアル環境モデルについて。オーサー環境（コンテンツの制作、編集、レビュー用）とパブリッシュ環境（コンテンツの公開時およびエンドユーザーがアクセス可能）を含みます。

### コンテンツ制作の課題

コンテンツ需要の高まりや、オムニチャネルコンテンツ配信の必要性、コンテンツのROIを向上させる重要性など、コンテンツ制作を拡大するための課題について認識している割合。 &#x200B;

### オーサリング

AEMの様々なオーサリングツールの概要です，

* **ページエディター** Web ページを作成および編集するための従来のWYSIWYG エディター。 &#x200B;
* **コンテンツフラグメントエディター**&#x200B;別のチャネルで再利用できるヘッドレスコンテンツを作成する。 &#x200B;
* **ユニバーサルエディター** AEM コンテンツと適切に実装された外部コンテンツの両方をサポートする、コンテンツ作成用の最新のUI。 &#x200B;
* **ドキュメントベースのオーサリング** Microsoft WordやGoogle Docsなどの使い慣れたツールをAEM ワークフローと統合してコンテンツを作成できます。 &#x200B;

### コラボレーション機能

AEMの共同作業機能には、同時審査、インラインレビュー、プロジェクトとワークフローの管理、タスクの委任などがあり、コンテンツ制作とレビュープロセスを合理化できます。

### 翻訳機能

AEMは、翻訳ワークフロー、言語コピー、ローカライゼーションツール、複数のサイトや言語を管理するマルチサイトマネージャーなど、多言語コンテンツ管理をサポートしています。

### Edge 配信サービス

AEM Cloudの一部としてエッジデリバリーサービスに言及し、コンテンツの制作と公開プロセスを強化。
