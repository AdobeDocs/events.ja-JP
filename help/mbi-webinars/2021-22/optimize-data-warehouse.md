# Commerce Intelligence Data Warehouseの最適化

>[!NOTE]
>
>Adobe Commerce Intelligenceは以前[!DNL Magento Business Intelligence (MBI)]と呼ばれていました。 過去の出来事の記録は以前の名前を反映しています。

Data Warehouse Managerでは、テーブルと列の同期設定を管理し、テーブルのスキーマをドリルダウンして、レポートで使用する計算列を作成できます。

>[!NOTE]
>
>Data Warehouse Managerへのアクセスには、管理者権限が必要です。

このウェビナーの内容：

- Data Warehouseの概要
- 更新サイクルと最適化
- レプリケーションと再チェック
- Data Warehouse ビュー
- 質疑応答
