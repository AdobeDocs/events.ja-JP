# あらゆる場所で 1 回で配信：Adobe Dynamic Media によるオムニチャネルエクスペリエンスの拡張

AEM Assetsで画像とビデオの変換と最適化を効率化する方法について知りたいですか？

Adobe Experience Manager チャンピオンの Vengadesh Shanmugavelu が、Adobe Experience Manager Assetsでの Dynamic Media の使用について詳しく説明します。 デジタルコンテンツ作成の拡大・縮小に関する主要な要素を説明します。

>[!VIDEO](https://video.tv.adobe.com/v/3481545/?learn=on&enablevpops)

* Adobe Experience Manager Dynamic Media でプロファイル設定、スマートレンディション、リアルタイム最適化を使用してインテリジェントなメディア配信を実現する仕組み。
* AEM コアコンポーネントを通じて Dynamic Media をアクティブ化し、Dynamic Media Open API を使用して企業全体でその機能を拡張する方法。

マーケター、アーキテクト、開発者を問わず、このウェビナーでは、1 回で配信し、パフォーマンス、一貫性、制御を備えてどこにでも拡張する方法を説明します。

Vengadesh Shanmugavelu 氏は、Publicis Sapient のシニアマネージャーエンジニアリング担当です。 AEMおよび Interwoven 製品スイートのコンテンツ管理ソリューションアーキテクチャ、設計、開発、実装、トレーニング、サポートで 15 年以上の経験があり、AEMおよびAdobe marketing cloud ソリューションでは 7 年以上の経験があります。 アーキテクトとしてAdobe認定エキスパートであり、AEMのリードデベロッパーです。また、Adobe TargetのアーキテクトとしてAdobe認定エキスパートであり、Adobeの認定エクスペリエンスプラットフォームエンジニアです。 2022 年からAEMチャンピオンです。
