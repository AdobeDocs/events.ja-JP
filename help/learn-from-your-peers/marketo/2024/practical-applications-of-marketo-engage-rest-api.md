# 強力なワークフローオートメーションを実現：Marketo Engage REST APIの実用的なアプリケーション

Marketo Engage REST APIの機能を利用すれば、マーケティング業務のワークフローを合理化し、手作業を削減して、収益創出のための時間を確保できます。 REST APIを使用すると、Zapierなどの自動化ツールを活用して、自動化されたワークフローやPythonなどのプログラミング言語をトリガーし、アクションを一括で実行できます。

このウェビナーでは、次の手順に従います。

- Marketo Engage REST APIの概要、最初のAPI ユーザーの作成方法、様々なユーザー間でのAPI使用状況のモニタリング方法について説明します。
- ZapierなどのツールやPythonなどのプログラミング言語の中から、REST APIを使用する方法を説明します。
- REST APIを使用して、リードのマージ、UTM トラッキングプログラムの作成、アクティビティデータの抽出などの一般的なマーケティング業務ワークフローを合理化する方法の実際のユースケースにヒントを得ることができます。
