# AEM as a Cloud Service 2023.6.0 リリースの更新

* DAM 用の新しいユーザーインターフェイス
* 検索の機能強化
* 3D アセット管理
* メタデータ管理の強化
* AEM [!DNL Assets] + [!DNL Frame io] 統合
