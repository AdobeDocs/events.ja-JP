# AEM as a Cloud Service 2023.6.0 リリースアップデート


このリリースの機能では、Adobe Experience Manager [!DNL Assets] のイノベーションに重点を置いており、以下が含まれています。

* DAM の新しいユーザーインターフェイス
* 検索の機能強化
* 3D アセット管理
* メタデータ管理の機能強化
* AEM [!DNL Assets] + [!DNL Frame io] Integration
