# AEM リリースの概要ビデオ

このリリースの機能は、Experience Manager AssetsとCloud Serviceのイノベーションに焦点を当てており、次の機能が含まれています。

## AEM Assets

* コレクションをサポートするためのアセットセレクターの機能強化&#x200B;
* Assets Content Hub - サポート DRM （期限切れおよびライセンス済みのアセット）&#x200B;

## AEM Cloud Service

* Edge DeliveryのCloud Manager サポート&#x200B;
* Edgeサイドインクルード （ESI）&#x200B;
* 基本認証&#x200B;
* インタラクティブなAEM Developer Console
