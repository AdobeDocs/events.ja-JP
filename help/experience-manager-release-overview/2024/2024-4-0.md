# AEM as a Cloud Service 2024.4.0 リリースアップデート

このリリースの機能は、Experience Manager AssetsとCloud Serviceのイノベーションに焦点を当てています。

>[!VIDEO](https://video.tv.adobe.com/v/3429111/?learn=on)

このリリースの機能は、Experience Manager AssetsとCloud Serviceのイノベーションに焦点を当てており、次の機能が含まれています。

**AEM Assets**
* Assets ビューでアセット/フォルダー名を1回クリックします
* 自然言語を使用した検索体験の簡素化
* Assets ビューからのスマート切り抜きを含むダイナミックレンディションのプレビュー
* Adobe AEMのクイックアクション機能なら

**Cloud Manager**
* CDNの最適化と調整
* バージョンのパージと監査ログのパージのメンテナンスタスクのセルフサービス設定
* セルフサービスの専用IP サポートにより、Splunkを超えたベンダーにもログストリーミングを拡張できます
* ESIのCDN サポート、またSDI経由
* RDEのJava ロギング体験の向上
