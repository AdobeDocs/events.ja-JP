# 2024.8.0 AEM リリースの概要ビデオ

このリリースの機能は、AEM AssetsとDynamic Media、Forms、Sites、Cloud Foundationのイノベーションに重点を置いており、次の機能が含まれています。

* 1:1個の動的バリエーション用のDynamic Media コンポーザブルテンプレート
* Forms GenAI アシスタント
* AEMのコンテンツ構造を新たに統合したユニバーサルエディター&#x200B;
* Cloud Managerを介したEdge Delivery ServicesのDV証明書とサポートのライフサイクル管理
* エクスペリエンス監査ダッシュボード
* パイプラインのパフォーマンスの向上

>[!VIDEO](https://video.tv.adobe.com/v/3433381/?learn=on)

## 概要

ライブコピーやマルチサイトマネージャー機能の導入など、Adobe Experience Managerの最新情報を提供し、利用者がバリエーションを持つ複数の場所のコンテンツを簡単に管理できるようにしました。 &#x200B;らに、アセット管理向けの新しいFireflyの機能強化が示され、検索結果が見つからなかったときに新しいアセットを生成できるようになりました。 &#x200B;らに、ダイナミックメディアテンプレートを導入し、値が効率的に変更される複数のバナーを簡単に作成できるようになりました。 &#x200B;また、このイベントでは、AM フォーム用の新しいJenn-air システムが強調され、フォーム作成プロセスを加速させるアーリーアダプターのアクセスが提供されました。 &#x200B;最後に、プロジェクトのデプロイメントを効率化するために、edge delivery servicesのサポートやdomain validated DB証明書などのCloud Managerの機能強化を発表しました。 &#x200B;
