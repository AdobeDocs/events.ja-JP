# AEM as a Cloud Service 2024.3.0 リリースアップデート

このリリースの機能は、Experience Manager AssetsとCloud Serviceのイノベーションに焦点を当てています。

>[!VIDEO](https://video.tv.adobe.com/v/3428344/?learn=on)

今回のリリースでは、AI （人工知能）を活用し、インパクトの大きいエクスペリエンスの提供、より多くの従業員によるコンテンツ制作の支援という3つの柱が中心となります。 このリリースの機能には、次のものが含まれます。

**効果の高いエクスペリエンスの提供**

* このリリースは、オーサリングとEdge delivery servicesを組み合わせたもので、コンテンツの柔軟性とパフォーマンスの向上を実現します。
* 新しいAMP サイトは、フロントエンドスキルを使用して構築し、デプロイメントパイプラインを通じてデプロイできます。
* 既存のAMP サイトは、ページエディターでオーサリングを維持しながらフロントエンドをエッジ配信に移行することで、エッジ配信に移行できます。
* Edge配信サービスは、AM フォームでも利用でき、Microsoft Excelなどのツールを使用してフォームを迅速に構築およびデプロイできます。

**より多くのユーザーにコンテンツの作成を許可する**

* AMP Sitesの新しいビジュアルエディターであるユニバーサルエディターは、コンテンツ制作者が構造化コンテンツと非構造化コンテンツの両方を一貫して管理できる機能を提供します。
* コンテンツ作成者は、ページのコンテンツを視覚的に編集し、Edge Delivery Servicesに公開できます。
* また、ユニバーサルエディターでは、ヘッドレス実装など、Adobe Experience Managerの外部でホストされているコンテンツを視覚的に編集できます。

**AIを活用**

* AEM Sitesには新しい機能が追加され、生成AIを活用してパフォーマンスの高いコピーや画像のバリエーションを生成する、コンテンツバリエーションと呼ばれる機能が追加されました。
* コンテンツ制作者は、事前設定済みのテンプレートに値を入力することで、パフォーマンスの高い結果をすばやく得ることができます。
* コンテンツのバリエーションは、AMのコンテンツをテストし、最適化する実験機能と組み合わせることができます。

さらに、コンテンツフラグメントとモデルの拡張API サポートなど、開発者向けの機能強化により、コンテンツフラグメントの管理と統合を自動化しやすくなりました。 また、Cloud Managerには、専用のイングレス IP アドレス、柔軟なポートエグレス設定、VPN設定を可能にする、高度なネットワークセルフサービス設定オプションも用意されています。

その他の機能として、コンテンツフラグメントのバージョンを比較する機能、マルチサイト管理機能、エクスペリエンスフラグメントのサポート、コンテンツインポーターのアップデート、Sidekick拡張機能の強化などが挙げられます。

全体として、このリリースには、インパクトの大きいエクスペリエンスの提供、コンテンツ制作者の能力強化、AIの活用、開発者フレンドリーなツールと設定の提供を目的とした、さまざまな機能と機能強化が導入されています。
