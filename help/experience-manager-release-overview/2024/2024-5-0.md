# AEM as a Cloud Service 2024.5.0 リリースアップデート

このリリースの機能は，

* AEMおよびDynamic Mediaの新しい公開オプション
* ユニバーサルエディターの機能強化
* アダプティブ Forms基盤コンポーネントのコアコンポーネントへの移行
* Adaptive FormsでのCaptchaのその他のサポート
* Cloud Foundationの機能強化

>[!VIDEO](https://video.tv.adobe.com/v/3429503/?learn=on)

## 重要な留意点

* Adobe Experience Managerは、AI時代の顧客体験管理に重点を置いており、生成AIがインテントベースの体験を可能にし、企業がコンテンツ制作と体験をスケールアウトできるようにします。
* アセットのアップロードプロセスに新機能が追加され、ユーザーがアセットの公開方法を選択できるようになり、時間を節約し、クリック数を減らすことができました。
* ユニバーサルエディターには、エディターでページプロパティを直接設定する機能、スプレッドシート表示の機能強化、画像内のテキストにキャプションを付ける機能など、3つの新機能が追加されました。
* Adobe Content EverywhereとFormsには、基盤コンポーネントからコアコンポーネントに移行できる新しい移行ツールが導入されており、クロスチャネル目的でヘッドレスフォームを使用する機能が実現しました。
* Cloud Foundationは、迅速な開発環境、フロントエンドのテーマとテンプレートを活用する機能、ロギングの強化、コマンドラインインターフェイスの追加モードなど、さまざまな機能強化を受けています。 さらに、DDoS攻撃から保護し、攻撃の際には電子メールでユーザーに通知する対策も実施されています。
