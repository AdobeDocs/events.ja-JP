# 2024.11.0 AEM リリースの概要ビデオ

このリリースの機能は、Experience Manager Assetsとサイトのイノベーションに焦点を当てており、次の機能が含まれています。

## AEM Assets

* Assetsのインターフェイスの拡張性&#x200B;
* Content Hub Asset Renditionの可用性&#x200B;
* Dynamic Media：新しいレンディションパネル&#x200B;
* OpenAPI機能を備えたDynamic Media&#x200B;の高度な画像修飾子とプリセット&#x200B;
* Dynamic Media Delivery Report&#x200B;

## AEM Sites

* AEMのユニバーサルエディター&#x200B;Edge Delivery Servicesのアップデート
