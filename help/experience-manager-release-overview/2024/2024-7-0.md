# AEM リリースの概要ビデオ

このリリースの機能は、Experience Manager AssetsとCloud Serviceのイノベーションに焦点を当てており、次の機能が含まれています。

**AEM Assets**

* AEM AssetsのContent Credentials&#x200B;
* Dynamic Mediaを使用した動画のAIによるキャプション&#x200B;
* アップロード用のアセットセレクターの機能強化&#x200B;

**Cloud Manager**

* 顧客が管理するCDN資格情報のセルフサービス設定&#x200B;
* CDN パージ&#x200B;
* トラフィックフィルタールールアラート&#x200B;
