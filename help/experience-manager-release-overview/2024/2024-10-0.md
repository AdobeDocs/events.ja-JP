# 2024.10.0 AEM リリースの概要ビデオ

このリリースの機能は、Experience Manager AssetsとCloud Serviceのイノベーションに焦点を当てており、次の機能が含まれています。

## AEM Assets

* AEM AssetsとAssets UltimateおよびAsset Primeの新しい価格とパッケージ

## AEM Cloud Service

* ログをより多くの宛先に転送する
* ビジネスユーザー向けに、パイプラインのないURL リダイレクト&#x200B;
* RDEはConfig パイプライン YAMLをサポート&#x200B;ます。
* インタラクティブなAEM Developer Console
