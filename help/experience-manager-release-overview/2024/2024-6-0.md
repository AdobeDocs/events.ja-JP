# 2024.6.0 AEM リリースの概要ビデオ

このリリースの機能は、Experience Manager Assets、Sites、Cloud Foundationのイノベーションに重点を置いています。

* AEM AssetsとContent Hub
* OpenAPI機能を備えたDynamic Media
* Assets セレクターの機能強化 – タグと期限切れのアセット
* コンテンツフラグメント Admin Consoleのコンテンツフラグメントモデル
* Cloud Foundation - Cloud Managerでのプライベートリポジトリの追加
* クラウド基盤 – コンテンツの復元
