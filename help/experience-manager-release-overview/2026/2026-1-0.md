# 2026.01.0 AEM リリースの概要ビデオ

このリリースの機能は、Experience Manager Sites、Forms、Assets、Cloud Foundationのイノベーションに焦点を当てており、次の機能が含まれています。

* AEM Sites
* Edge Delivery アップデート

* AEM Assets
* AI 検索（多言語、セマンティック）

* AEM Forms
* Interactive Communications Associate UI

* AEM Cloud Foundation
* MCP サーバー
