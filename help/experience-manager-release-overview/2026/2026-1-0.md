# 2026.01.0 AEM リリースの概要ビデオ

このリリースの機能では、Experience Manager Sites、Forms、Assets、Cloud Foundation のイノベーションに重点を置いており、次のものが含まれています。

* AEM Sites
* Edge Deliveryの更新

* AEM Assets
* AI 検索（多言語、セマンティック）

* AEM Forms
* インタラクティブ通信関連付け UI

* AEM Cloud Foundation
* MCP サーバー
