# 2026.03.0 AEM リリースの概要ビデオ

>[!VIDEO](https://video.tv.adobe.com/v/3483061/?captions=jpn&learn=on&enablevpops)

このリリースの機能は、Experience Manager Sites、Forms、Assets、Cloud Foundationのイノベーションに焦点を当てており、次の機能が含まれています。

* サイト
   * Experience Production Agentのアップデートとローンチのサポート

* フォーム
   * IC エディターのワークフローのサポート
   * コアコンポーネントのテーマエディターのサポート
   * Formsのバージョン管理のサポート

* アセット
   * AEM Content Advisor - Adobe アプリケーションとの統合
   * Microsoft Office コネクタ （PowerPoint、Word、Excel）

* クラウド基盤
   * Governance Agent: ブランド管理
   * タッチ UIに対する監査権限
   * OIDC ログアウト機能
   * メンテナンス更新を一時停止

* Cloud Manager
   * Cloud Manager MCP Server
