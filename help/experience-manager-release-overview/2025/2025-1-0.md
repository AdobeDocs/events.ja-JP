# 2025.01.0 AEM リリースの概要ビデオ

このリリースの機能は、Experience Manager Sites、Forms、Assets、Cloud Foundationのイノベーションに焦点を当てており、次の機能が含まれています。

* AEM Sites
   * コンテンツフラグメントのコメント

* AEM Forms
   * AEM Formsによるインタラクティブなコミュニケーション

* AEM Assets
   * Dynamic Media テンプレート
   * Dynamic Media配信レポート

* AEM Cloud Foundation
   * Java 17または21でビルド
