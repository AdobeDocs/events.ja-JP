# 2025.5.0 AEM リリースの概要ビデオ

このリリースの機能は、Experience Manager Sites、Forms、Assets、Cloud Foundationのイノベーションに焦点を当てており、次の機能が含まれています。

## AEM Sites

* OpenAPIによるコンテンツフラグメント配信
* AEM コンテンツフラグメントはAJOで再利用されます
* AEM 6.5 LTS - Java 21のサポート

## AEM Assets

* スマートタグの強化
* AEM Dynamic MediaとAJO B2Cのネイティブ統合
* Content Hubの機能強化
* Figma ターンキーコネクタ用AEM Assets

## AEM Forms

* Forms Experience Builder
* Marketo コネクタ – フェーズ 2
