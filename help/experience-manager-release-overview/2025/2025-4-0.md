# 2025.04.0 AEM リリースの概要ビデオ

このリリースの機能は、Experience Manager Sites、Forms、Assets、Cloud Foundationのイノベーションに焦点を当てており、次の機能が含まれています。

## AEM Sites

* コンテンツモデル管理UI
* コンテンツフラグメントがUIとAPIを起動
* Edge Delivery Services &#x200B;とCommerceの統合

## AEM Cloud Foundation

* 認証されたOpenAPI ベースのAPIのサポート

## AEM Forms

* EDGE DELIVERY ベースのFormsのFM サポート
* ルールエディターUIで壊れたルールをハイライト表示
