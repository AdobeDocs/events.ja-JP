# AEM リリースの概要ビデオ

このリリースの機能は、インテリジェンス、アクセラレーションアクティベーション、クラウド基盤によるコンテンツ生成に重点を置いており、次の機能が含まれています。

## AEM Sites

* コンテンツフラグメントの自動タグ付け
* コンテンツフラグメントのリモートアセットのサポート  編集者

## AEM Sites、Assets、Forms

* 翻訳管理API

## クラウド基盤

* AEM 6.5 LTS （長期サポート）
* Edge Delivery Services 99.99へのSLA レポートの拡張
* ワンクリックでEdge Delivery Services サイトを作成
