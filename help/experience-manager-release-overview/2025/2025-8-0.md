# 2025.08.0 リリースの概要ビデオ

このリリースの機能では、Experience Manager Sites、Forms、Assets、Cloud Foundation のイノベーションに重点を置いており、次のものが含まれています。

## Cloud Foundation

* AEM CMK への安全な接続と自動処理を提供して、お客様のセキュリティと信頼を強化&#x200B;ます。
* パブリッシュ向け実稼動 OIDC - Jira
* 独自の Git の導入（BYOG）

## フォーム

* ルール作成を容易にすることで、作成者の効率を向上
* DoR をダウンロードするための OOTB ボタン

## サイト

* 自分のマークを表示する
* コンテンツセキュリティポリシー
* 独自の Git を用意
* スナップショットとレビュー
