# 2025.08.0 リリースの概要ビデオ

このリリースの機能は、Experience Manager Sites、Forms、Assets、Cloud Foundationのイノベーションに焦点を当てており、次の機能が含まれています。

## クラウド基盤

* AEM CMKに安全な接続と自動化を提供することで、お客様のセキュリティと信頼性を向上させます&#x200B;
* パブリッシュ向け製品情報OIDC - Jira
* 独自のGitを取り込む（BYOG）

## フォーム

* ルール作成を容易にし、作業効率を向上
* DoRをダウンロードするためのOOTB ボタン

## サイト

* 自分のマークを上げる
* コンテンツセキュリティポリシー
* 独自のGitを持ち込む
* スナップショットとレビュー
