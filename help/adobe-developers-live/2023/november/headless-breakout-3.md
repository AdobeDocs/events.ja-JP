# Adobe Developers Live Headless Breakout 3 - 2023年11月

Iryna Lagno氏とValerii Naida氏が、2023年11月6日にライブ録画されたこのセッションで、Adobe Experience Managerのコンテンツ管理に対する革新的なアプローチを紹介します。 Google Places APIなどの外部データソースをコンテンツフラグメントに統合する方法、およびCanvas Data APIとカスタム UI要素を見つける方法について説明します。 このセッションでは、ワークフローを合理化し、データ主導の要素をコンテンツ戦略に導入したい開発者やコンテンツマネージャーに役立つ知識を提供します。

[Experience League Communities](https://adobe.ly/48Rl57B)でやり取りを続けます。
