# AEM Universal Editor の概要

ユニバーサルエディターは、次世代のAEMコンテキスト内ページエディターで、AEMページエディターまたはSPAエディターの現在の制限を解決します。 フレームワーク、レンダリングエンジン、データソースとは独立しており、開発者が求めるあらゆる自由を提供し、真に普遍的なものにしています。
