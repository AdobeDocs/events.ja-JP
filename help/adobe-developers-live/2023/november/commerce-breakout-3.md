# CommerceBreakout 3 - 2023年11月

2023年11月6日にAdobe Developers Liveでライブ録画されたこのセッションでは、Kanika GeraとSurya Lamechが、Adobe CommerceとAWSの間のイベント駆動型インテグレーションを構築するプロセスについて話し合います。 Adobe Commerceの拡張可能なモデルや、Amazon EventBridgeとの連携の詳細、クラウドネイティブのデジタルコマース環境における実際のアプリケーションをご紹介します。 このセッションでは、最新のクラウドテクノロジーを活用した、拡張性と柔軟性の高いコマースソリューションの構築について解説します。

[Experience League Communities](https://adobe.ly/3ts1NW5)でやり取りを続けます。
