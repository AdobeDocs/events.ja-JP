# CommerceBreakout 1 - 2023年11月

2023年11月6日にリリースされた、Adobe Developers Liveの啓発セッションでFahd Siddiquiが参加し、Adobe Commerceの革新的なカタログおよび価格インデックス作成サービスが、B2B デジタルコマースをどのように変革しているのかを解説します。 商品アセット管理の課題と、Adobe Commerceが提供する迅速で効率的なショッピング体験の提供についてご紹介します。 このセッションは、B2B コマーステクノロジーのダイナミックな世界において、専門知識を強化したい開発者にとって非常に重要です。

[Experience League Communities](https://adobe.ly/3rJfZcN)でやり取りを続けます。
