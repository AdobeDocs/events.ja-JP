# APIを使用して、最新のAssets ピッカーを他のUI/製品に組み込み、サードパーティ製アプリケーションと統合する方法

APIを使用して、最新のAssets ピッカーを他のUI/製品に組み込み、サードパーティ製アプリケーションと統合する方法

アセットを他のアプリケーションに統合する際に、あらゆるオプションからメリットを得る方法について説明します。 AEM Assets Cloud ServicesをCMSに統合することは、困難を伴いますか？ AEM Assetsの安定したテスト可能な統合機能を数時間や数日ではなく数分で作成する方法を知りたいですか？ 「Destination Selector」と「Asset Selector」の2つのReact プロジェクトを、モジュール形式で独立した簡単に実装できるフロントエンドコンポーネントに変換した経験を通じて、マイクロフロントエンドのパワーと汎用性を実証します。 →次の場所で会話を続けます：https://adobe.ly/3LqELUM
