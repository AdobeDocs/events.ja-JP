# Adobe Sign API を使用したドキュメントの生成とアプリ内での電子署名の取得

Adobeドキュメント生成 API は、Microsoft Word テンプレートをデータと結合して駆動する、強力なドキュメント作成サービスです。 Adobe Sign API と組み合わせると、開発者は署名ワークフローで処理できる動的ドキュメントを簡単に生成できます。
