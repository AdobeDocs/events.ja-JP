# Adobe Sign API を使用したドキュメントの生成とアプリでの電子サインのキャプチャ

Adobeドキュメント生成 API は、データと結合されたMicrosoft Word テンプレートによって駆動される強力なドキュメント作成サービスです。 Adobe Sign API と組み合わせると、Sign ワークフローを使用して、処理する準備が整った動的なドキュメントを簡単に生成できます。
