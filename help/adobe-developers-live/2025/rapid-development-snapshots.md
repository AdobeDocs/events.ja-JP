# 凍結、復元、革新：クラウド開発のための迅速な開発環境スナップショットを使用した実践的なデモ

Adobe Experience Manager Cloud Serviceの迅速な開発環境（RDE）によって、いかにクラウドネイティブな反復が可能になるかをご確認ください。 スナップショット機能を使用して、開発者がRDEの状態を完全にフリーズ、復元、再利用できるようにする方法について説明します。 これにより、Adobe I/O CLI RDE プラグインを使用して、テストを効率化し、即座にロールバックして、開発サイクルを加速させることができます。 クラウド開発の効率を高め、コーポレートガバナンスポリシーに準拠。

>[!VIDEO](https://video.tv.adobe.com/v/3477705/?learn=on&enablevpops)

このセッション「Freeze, Restore, Innovate: Hands-On with Rapid Development Environment Snapshots for Cloud Development」では、Adobeのソフトウェア開発エンジニアであるRemo Liechti氏が出演し、Adobe Developers Live 2025で事前録画されました。 Remoは、迅速な開発環境（RDE）がAdobe Experience Manager Cloud Serviceでクラウドネイティブな迅速なイテレーションをどのように実現し、新しいスナップショット機能によってRDEの状態を完全に凍結、復元、再利用できるのかを示します。 Adobe I/O CLI RDE プラグインを使用して、AEM Cloudでテストを効率化し、即座にロールバックし、開発サイクルを高速化する方法を説明します。

Adobe Developers Live 2025を支援してくださったスポンサーのAlgoliaとEnsembleに感謝いたします。
