# 凍結、復元、革新：迅速な開発環境での実践クラウド開発用のスナップショット

Adobe Experience Manager Cloud Serviceの迅速な開発環境（RDE）によって、クラウドネイティブの迅速な反復がどのように可能になるかを説明します。 スナップショット機能を使用して、開発者が完全な RDE 状態を凍結、復元、再利用する方法について説明します。 これにより、Adobe I/O CLI RDE プラグインを使用して、テストとロールバックを即座に合理化し、開発サイクルを高速化できます。 クラウド開発の効率を高め、コーポレートガバナンスポリシーと連携します。

>[!VIDEO](https://video.tv.adobe.com/v/3477705/?learn=on&enablevpops)

今回のセッションでは、Adobeのソフトウェア開発エンジニアである Remo Liechti 氏を特集し、Adobe Developers Live 2025 の事前収録を行いました。 Remo は、迅速な開発環境（RDE）がAdobe Experience Manager Cloud Serviceで高速なクラウドネイティブなイテレーションを可能にする仕組みと、新しいスナップショット機能によって開発者が完全な RDE 状態を凍結、復元、再利用できるようにする仕組みを実演します。 Adobe I/O CLI RDE プラグインを使用して、AEM Cloud でのテストを合理化し、即座にロールバックして、開発サイクルを高速化する方法について説明します。

Adobe Developers Live2025 にご協賛いただいたスポンサーのアルゴリアとアンサンブルに心より感謝申し上げます。
