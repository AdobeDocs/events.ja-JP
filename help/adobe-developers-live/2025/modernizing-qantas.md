# Edge Delivery Servicesを使用したQantas.comの最新化（EDS）

カンタス航空が、Adobe Experience Manager Edge Delivery Servicesを活用してweb サイトをどのように刷新したかをご確認ください。 この変革により、公開時間を数週間から数分に短縮し、開発を簡素化し、よりクリーンなセマンティックマークアップによりSEOを向上させることができました。 オンプレミス導入からエッジアーキテクチャへの移行について学び、JavaScriptベースのワークフローで人材プールをどのように拡大したかをご確認ください。 この最新のアプローチが、オーサリングのスピードをどのように向上させ、企業ガバナンス目標をどのようにサポートしているのかをご確認ください。 Adobe Adobeに関するイベントを確認し、より詳細なインサイトを獲得しましょう。

>[!VIDEO](https://video.tv.adobe.com/v/3477681/?learn=on&enablevpops)

このファイアサイドチャット — Edge Delivery Servicesを使用したQantas.comの近代化 – は、Accenture SongのソリューションアーキテクトであるTim Lavelleと、AdobeのシニアプロダクトスペシャリストであるGlen LawsonがSydneyからライブで録画したものです。 カンタスの大規模な再構築プロジェクトとAdobe Experience Manager Edge Delivery Servicesへの移行について話し合い、オンプレミス導入からエッジアーキテクチャへの移行によって、公開にかかる時間を数週間から数分に短縮した方法を解説します。 この最新のアプローチによって、開発が簡素化され、JavaScriptベースのワークフローで人材プールが拡大し、よりクリーンなセマンティックマークアップとユニバーサルエディターによってオーサリングのスピードとSEOが向上した方法を紹介します。
