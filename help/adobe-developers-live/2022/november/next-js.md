# Next.JS アプリにインコンテクストおよびヘッドレスオーサリングを導入

Next.jsは、特にハイブリッド静的レンダリングとサーバーレンダリングを使用して、本番環境に対応したReact アプリを構築するための優れた開発者エクスペリエンスを提供します。 Adobe Experience Managerを活用したサイトに最適な配信機能です。

このセッションでは、次の方法について解説します。

* Adobe Experience Managerのコンテンツを使用して、新しいNext.js アプリをすばやくブートストラップできます
* React Editable SDKを使用して、インコンテクスト編集を有効にする
* Adobe Experience Manager ヘッドレス CMSからコンテンツを取得し、Next.js アプリでレンダリングする
