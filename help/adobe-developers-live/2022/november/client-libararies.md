# 機能するForms - クライアントライブラリとヘッドレスFormsの活用

Adobe Experience Manager Formsは、Cloud Serviceに多くのエキサイティングな新機能をリリースしました。 フォーム開発者の場合、Adobe Experience Manager Formsのユーザーエクスペリエンスを向上させるために、カスタムスクリプトやサードパーティのJavaScript ライブラリを活用する必要がある場合があります。

このセッションでは、クライアントライブラリを使用し、フォーム作成者の手からスクリプトを取り出す方法を紹介します。高度な計算、日付操作、カスタムモーダルウィンドウの実行に必要な機能を簡単に選択できます。

また、ヘッドレスアダプティブFormsを使用した最新のイノベーションも紹介します。ヘッドレスアダプティブでは、一度作成したフォームを、webやSPA アプリなどの複数のチャネルで使用できます。reactを使用したフォームの作成や、react アプリのカスタマイズも可能です。

アドビの製品エキスパートが、WhatsAppなどのインタラクティブなチャネルで同じフォームを使用して、データキャプチャを実現する方法を解説します。これにより、フォームを思い通りに動作させることができます。
