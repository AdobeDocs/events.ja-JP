# トップ 3 スペシャルエディション！ 私の仕事を見る – 3 つの方法 [3 番目のビデオ ]

>[!VIDEO](https://video.tv.adobe.com/v/3470074/?learn=on&enablevpops)

ワーカーとリクエスターがWorkfrontでワークフローを最適化するのに役立つ上位 3 つの機能について説明します。 ドラフトリクエスト、カスタムフォームビュー、ボードを使用して、生産性を向上させ、プロセスを合理化する方法について説明します。 これらのツールは、タスクを効率的に管理し、視覚的に共同作業し、大規模なプロジェクト内で目標を追跡するために不可欠です。 これらの機能でWorkfrontの作業がどのようにサポートされ、エクスペリエンスがどのように向上するかについて説明します。
