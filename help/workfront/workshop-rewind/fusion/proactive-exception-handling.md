# Workfront Fusionのプロアクティブな例外処理フレームワーク

Adobe Workfront Fusionのプロアクティブな例外処理によって、自動化プロセスをどのように変革できるかをご確認ください。 このフレームワークは、エラーを効率的に特定、追跡、解決し、混乱を最小限に抑え、業務パフォーマンスを向上させるのに役立ちます。 自動化の健全性をリアルタイムで把握し、一元化されたエラーレポートを活用し、プロセスを最適化してビジネス成果を向上させることができます。 このビデオワークショップでは、Workfrontの機能を活用して例外管理を強化するためのインサイトを提供します。
