# Workfront Fusion と JSONata を使用した JSON の作成

Adobe Professional Servicesの Steve Watts と Brian Haight がパートナーとしてホストしたこのセッションでは、JSON パス表記の読み取り、データタイプの理解、Fusion での適切なスタートパスの選択に関する基本について説明しました。

また、データの検索とフィルタリング、JSON のリシェイプ、その場での高度な変換の実行など、3 つの実際の例についても説明しました。 時間を節約し、データ処理を簡素化したいと考えている Fusion 管理者にとっては、必須のスキルです。
