# ネイサンの5つのFaves - Workfrontワールドの基本API

Workfront APIが、お客様のビジネスにどのように役立つのかをご案内します。 Adobeテクニカルサポートエンジニア IIIのNathan Johnson氏が、APIの基本を詳しく説明し、実際の世界に導きます。

このセッションでは、NathanがWorkfront APIの基本、概要、仕組み、重要な理由を説明した後、実際の機能を紹介する5つの好きなユースケースを紹介します。

>[!VIDEO](https://video.tv.adobe.com/v/3492587/?learn=on&enablevpops)

このセッションでは、Adobe Workfront APIの基礎知識から実際のアプリケーションに至るまで、Adobe Experience Manager APIを活用して日々の業務を進めるエキスパートが、APIを活用した分かりやすいインサイトを提供します。 APIの初心者にとっても、技術的なツールキットの拡張を検討している場合でも、このセッションは、コンセプトと機能の間の点を結びつけるのに役立ちます。
