# Workfront Data Connectの強力な機能を活用

## 概要

* [Data Connect](https://experienceleague.adobe.com/en/docs/workfront/using/reporting/data-lake/data-lake-overview)とは何ですか？ Workfrontのデータを、Tableau、Power BI、Domo、Lookerなどのビジュアライゼーションツールに接続することで、ビジネスインサイトを提供する安全で拡張性の高いソリューションです。
* これはUltimate パッケージに含まれていますが、PrimeおよびSelectのアドオン（$）として利用できます。 詳細については、アカウントチームにお問い合わせください。
* 少なくとも今すぐ試すテストドライブは提供できませんが、アカウントチームがいつでも連絡を取り、パーソナライズされたデモでサポートできるかどうかを確認することは可能です。
* **Experience League ドキュメント**&#x200B;には、[用語マッピングテーブル &#x200B;](https://experienceleague.adobe.com/en/docs/workfront/using/reporting/data-lake/data-dictionary)が用意されており、Workfront インターフェイス名、API リファレンス、データレイクフィールド名の変換に役立ちます。
* **現在の機能** Data Connectは、現在110以上のWorkfront オブジェクト（10月の起動時の70から増えました）をサポートしており、データは4時間ごとに更新されます。 ビューには、現在の状態、日次の履歴スナップショット、履歴分析用のトランザクションイベントデータの3つのタイプがあります。
* **お客様のユースケース**&#x200B;お客様は、ユーザーのアクティビティの監視、プロセスのボトルネックの特定、ステータス時間の追跡、コンプライアンスチェック、SLA トラッキング、キャパシティプランニング、既存のデータウェアハウス内の外部データセットとのWorkfront データの結合などにData Connectを使用しています。
