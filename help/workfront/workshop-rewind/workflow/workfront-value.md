# ピッチパーフェクト：Workfrontの価値を定義し、伝える

「理由」の定義から勝利のピッチの提供まで、 Workfront インスタンスは毎日価値を提供しますが、そのストーリーを、ふさわしい評価、サポート、リソースを得られる形で共有していますか？

ワークショップでは、スライドを組み合わせてカスタマイズ可能な資料を作成しました。この資料は、お客様の組織でWorkfrontが「理由」を特定し、結果を報告し、エグゼクティブ、同業者、エンドユーザーの共感を得られる方法で成果を伝えるのに役立ちます。
