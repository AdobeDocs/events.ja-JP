# Pitch Perfect:Workfrontの価値の定義と伝え方

「なぜ」を定義することから、勝利のピッチを提供することまで！ Workfrontインスタンスは、日々の業務で価値を提供していますが、そのストーリーを共有することで、自社が求める認知、サポート、リソースを得ることができますか？

ワークショップでは、スライドを通じてカスタマイズ可能なデッキを作成し、組織のWorkfrontの「なぜ」、結果の報告、経営陣、同僚、エンドユーザーの共感を呼ぶような方法での成果の報告を支援しました。
