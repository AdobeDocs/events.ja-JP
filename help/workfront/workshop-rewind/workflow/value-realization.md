# Workfrontの価値を最大化

組織でAdobe Workfrontの価値を最大限に高める方法を説明します。 このビデオでは、目標と結果の関連付け、主要業績評価指標（KPI）の特定、ベースラインの取り込み、結果の測定、関係者との結果の共有について説明します。 Workfrontがビジネス目標、課題、成功に与える影響を明らかにするライブドキュメントの作成方法を説明します。 これらの戦略を適用することにより、Workfrontがビジネスニーズを満たし、投資に対する正当性を提供できるようにします。 ビデオを視聴してAdobeのエキスパートからインサイトを得て、Workfrontのプラクティスを強化しましょう。
