# Workfrontの価値を最大化

Adobe Workfrontの価値を最大化する方法をご紹介します。 この動画では、目標と成果の調整、重要業績評価指標（KPI）の特定、ベースラインの取得、成果の測定、関係者との結果の共有などについて説明します。 Adobe Workfrontがビジネス目標、課題、成功に与える影響を明らかにする実践的なドキュメントを作成する方法をご覧ください。 これらの戦略を適用することで、Workfrontがビジネスニーズを満たし、投資の正当化に利用できることを確認できます。 動画でAdobeの専門家から得たインサイトを活用し、Workfrontのプラクティスを強化しましょう。
