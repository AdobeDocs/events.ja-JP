# 統合承認について – 方向性、レポート、次期機能

統合承認は、Workfrontにおける承認の未来を表します。 Adobeのプロダクトマネジメント部門では、Workfrontで承認するための方向性や、現在のプルーフとの違い、長期的な移行の展望について有益なセッションを提供しています。

>[!VIDEO](https://video.tv.adobe.com/v/3481834/?learn=on&enablevpops)

セッションを逃した場合、心配はありません！ [ スライドデッキ ](https://workfront-experience.s3.us-west-2.amazonaws.com/Training/Guides/Customer+Success+at+Scale/032626+-Understanding+Unified+Approvals.pdf)を確認し、オンデマンド録画を視聴します。

毎月新しいイベントが追加されるので、最新のセッションについては、[Experience League イベント ページ ](https://experienceleague.adobe.com/ja/events)を必ず確認してください。
