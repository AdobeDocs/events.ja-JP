# よりスマートにキャプチャし、グローバルに拡張 – WorkfrontでのカスタムFormsとローカライズ

アドビのAEO （Advanced Enterprise Operations） Capabilities イベントシリーズの第2 セッションでは、AdobeのプロダクトマネージャーであるKhachatur SedrakyanとKristine Tumanyanが、より動的で自動化されたデータ精度の高い受入体験を実現する、4つの新しい高度なロジックタイプ、表示、編集可能性、条件付きフォーマット、デフォルト値について解説しました。

さらに、ユーザーが入力したラベルのカスタム翻訳を追加できる新しいローカライゼーションの拡張機能を導入し、グローバル企業が、より一貫性のあるアクセス可能な、地域に合わせた体験をWorkfront全体で構築するのに役立ちました。
