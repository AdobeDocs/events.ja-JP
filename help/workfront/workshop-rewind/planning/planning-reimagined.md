# Workfront計画、再構築：SNHUのCreativeユースケース

Southern New Hampshire Universityのマーケティングオペレーション担当アソシエイトバイスプレジデントであるHeather MacDonald氏が、SNHU チームがすぐに利用できる（OOTB）課題に対処し、Workfrontプランニング機能をクリエイティブで予期せぬ方法で活用して、マーケティングオペレーションにおける可視性、連携、意思決定を促進した方法を解説します。 ユニークなポートフォリオの設定から独創的なタイムライン戦略まで、彼女はあなたが自分の環境で適用できる実際の例と実用的なヒントを提供しました。
