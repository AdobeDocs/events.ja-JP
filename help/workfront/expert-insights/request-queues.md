# Expert Insights - Lyndsy Denkのリクエストキュー

Expert Insightsへようこそ！  あなたの「オンデマンド」Workfront CSMのシンシア・ブーンです。このエピソードでは、Workfrontの有名人[@Lyndsy-Denk](https://experienceleaguecommunities.adobe.com/t5/user/viewprofilepage/user-id/17573167?profile.language=ja)が登場します。 ここでは、LyndsyのWorkfront Storyについて解説した後、ヘルプデスクのリクエストキューを詳しく説明します。

>[!VIDEO](https://video.tv.adobe.com/v/3465272/?learn=on&enablevpops)

チャットの中で、Lyndsyは，

* ヘルプデスクのリクエストキューを作成するためのヒント
* 彼女が作業を整理し、リクエストを追跡する方法
* リクエストを管理およびルーティングする方法
* 素晴らしいキュー分析ダッシュボードもデモしています！

## 講演者について 

Lyndsyは、IDEXXのマーケティングトレーニングおよびイネーブルメントマネージャーです。 Workfrontに加え、DAMの管理者として、マーケティング部門の社内コミュニケーションを担当しています。 テクニカルライティング、インストラクショナルデザイン、高等教育、高校での競争的なスピーチやディベートのコーチングのバックグラウンドを持っています。

彼女は夫と一緒にメイン州南部に住んでおり、今までで最高の猫と一緒に抱き合い、ポッドキャストを聴きながら携帯電話でダムゲームをプレイし、現実を逃れるためにフィクションを読み、ポートランドでおいしい食べ物に多額のお金を費やしています。
