# Expert Insights - Reporting Practices with Nichole Vargas

このエピソードは、私の同僚であるNichole Vargas氏からインスピレーションを得たもので、Workfront スケール チームとして知られる素晴らしい「3人のマスケット銃兵」の1人です。

記録として、彼女は私たちのイベントと同じくらい素晴らしく、インテリジェントで、寛大です！ 

オンデマンドWorkfrontのCSMである私と一緒に、Nicholeがレポートの達人になった経緯、彼女のお気に入りの機能とヒント、そして私たち全員が「Nicholeのようになる」方法について話します。 

>[!VIDEO](https://video.tv.adobe.com/v/3469966/?captions=jpn&learn=on&enablevpops)

## ヒント

* 諦めるな。 好奇心を持ち、失敗を恐れずに挑戦し続けましょう。 
* リソースを活用しましょう！ Experience Leagueを利用して、イベントを拡張し、さらに重要なのは、Adobe Workfrontの仲間です。 
* レポートを送信した後に必要な予定の次のステップまたはアクション項目を共有します。 メッセージにコンテクストと深みを加える。

* 独自のリソースを作成して、レポートノート、テキストモード、便利なリンク（メモページドキュメント、スプレッドシート、またはWorkfront プロジェクト）を保存します。 
* レポートで伝えたい「大局的」なメッセージやストーリーについて考え、リーダーシップやユーザーと共有したい感情的なメッセージを考慮することを忘れないでください。 
* 利用者との関係を維持し、その仕組みについてもつながりましょう。 レポートでデータを促進しているので、最も正確な画像をレポートしていることを確認してください。
