# データドリップ：古いシステムから Web SDKに移行するメリット

この Data Drip ウェビナーに参加しましょう。Adobe カスタマーテクニカルアドバイザーの Arindam Agarwal が、Adobe Web Software Development Kit の全身的なメリットを紹介します。 Adobe Web SDKは、Adobe Experience Platform Edge Networkを通じてサービスを活用するクライアントサイドソリューションであり、持続的な競争上の利点を生み出し、運用コストを削減し、古いライブラリと比較してパフォーマンスを向上させます。 Arindam は、Web SDKの機能と利点を詳しく説明します。例えば、– 実装の容易さ – 統合の簡素化 – よりスマートなトラッキングの実現 – 負債の軽減 – イベントサイド転送 – Web サイトトラフィックフローなどです

>[!VIDEO](https://video.tv.adobe.com/v/3476271/?learn=on&enablevpops)

## Adobe Web SDKの機能のロック解除

Adobe Web SDKへの移行がデジタルデータの収集とパーソナライゼーションをどのように変革するかについて説明します。 この最新のアプローチにより、ワークフローが合理化され、組織での将来的なプルーフが可能になります。

* **Unified Data Collection** 複数のライブラリを 1 つの軽量のSDKに置き換えることで、ページの重さとメンテナンスを軽減します。
* **Edge Network統合** 分析、パーソナライゼーション、およびサードパーティプラットフォームへの効率的な配信を実現するために、Adobeの分散サーバーを介してすべてのデータをルーティングします。
* **パフォーマンスとコンプライアンス** より高速な Web サイト、ガバナンスの向上、同意管理などの組み込みのプライバシー機能をご利用いただけます。
* **実装の簡素化** 明確な計画、スキーママッピング、堅牢なデバッグツールを使用して移行します。

これらの機能を活用することで、よりクリーンなデータ、より強力なコンプライアンス、拡張性の高いデジタルエクスペリエンスを実現できます。

## 統合データ収集の説明

Adobe Web SDKは、複数のトラッキングライブラリ（Analytics、Target、Audience Manager）を 1 つの統合された alloy.js スクリプトに統合します。 この単一のSDKは、

* エッジネットワークへの 1 回の呼び出しですべてのデータを送信するので、ネットワークリクエストとページ読み込み時間が短縮されます。
* データ収集とパーソナライゼーションの両方を処理し、個別のリクエストを不要にします。
* Adobe Edge Network と統合され、設定済みのソリューションやサードパーティの宛先にデータをルーティングします。
* は、標準化されたエクスペリエンスデータモデル（XDM）スキーマをサポートし、プラットフォーム間で一貫したデータを生成します。

このアプローチにより、メンテナンスを簡素化し、パフォーマンスを向上させ、チーム間のシームレスなコラボレーションを可能にします。
