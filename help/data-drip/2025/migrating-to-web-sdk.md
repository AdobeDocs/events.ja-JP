# Data Drip：旧システムからWeb SDKへの移行のメリット

AdobeのカスタマーテクニカルアドバイザーであるArindam Agarwal氏が、Adobe Web Software Development Kitのシステミックなメリットについて解説します。 Adobe Web SDKは、Adobe Experience Platform Edge Networkを通じてサービスを活用するクライアントサイドのソリューションです。これにより、競争力を維持しながら、運用コストを削減し、従来のライブラリと比較してパフォーマンスを向上できます。 Arindamは、Web SDKの詳細な概要を提供し、その特徴と利点を次のように説明します。 – 実装の容易さ – 簡単な統合 – スマートなトラッキングの有効化 – 賠償責任の軽減 – イベント側転送 – Web サイトのトラフィックフロー

>[!VIDEO](https://video.tv.adobe.com/v/3476271/?learn=on&enablevpops)

## AdobeのwebSDKの能力を引き出す

Adobe Web SDKへの移行が、デジタルデータの収集とパーソナライゼーションをどのように変革するのかをご確認ください。 この最新のアプローチにより、ワークフローが合理化され、組織の将来性を確実に保つことができます。

* **統合データ収集**&#x200B;複数のライブラリを1つの軽量SDKに置き換えて、ページの重みとメンテナンスを軽減します。
* **Edge Networkとの統合** Adobeの分散サーバーを介してすべてのデータをルーティングし、分析、パーソナライゼーション、サードパーティのプラットフォームに効率的に配信します。
* **パフォーマンスとコンプライアンス** web サイトの高速化、ガバナンスの強化、同意管理などの組み込みのプライバシー機能を利用できます。
* **簡素化された実装**&#x200B;明確なプランニング、スキーママッピング、堅牢なデバッグツールを使用した移行。

これらの機能を活用することで、よりクリーンなデータ、より強力なコンプライアンス、スケーラブルなデジタルエクスペリエンスを実現できます。

## 統合データ収集の説明

Adobe Web SDKは、複数のトラッキングライブラリ（Analytics、Target、Audience Manager）を1つの統合されたalloy.js スクリプトに統合します。 このシングルSDK:

* 1回の呼出しであらゆるデータをエッジネットワークに送信し、ネットワーク要求とページの読み込み時間を削減します。
* データ収集とパーソナライゼーションの両方を処理するため、別々のリクエストをする必要がなくなります。
* Adobeのエッジネットワークとの統合：設定されたソリューションやサードパーティの宛先にデータをルーティングします
* 標準化されたExperience Data Model （XDM）スキーマをサポートし、プラットフォームをまたいで一貫性のあるデータを収集。

このアプローチにより、メンテナンスの簡素化、パフォーマンスの向上、チーム間のシームレスなコラボレーションが可能になります。
