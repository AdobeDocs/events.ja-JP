# データドリップイベント

Data Drip シリーズは、Adobeのエキスパートが集うシリーズで、あらゆるスキルレベルを持つ既存のAdobe AnalyticsおよびTargetのお客様を歓迎します。 このシリーズでは、イベントのトピックを反映した最新の機能とベストプラクティスのいくつかに焦点を当てます。 続いて、製品内の機能を利用してベストプラクティスを実行する方法の例を、ライブデモやスクリーンショットで紹介します。 顧客は、製品が提供するものを最大化しながら、競争力を維持し、成功を達成する方法をより深く理解して離脱することが保証されます。
