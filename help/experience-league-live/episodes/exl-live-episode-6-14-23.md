# Web チャネルとコンテンツ実験を使用したAdobe Journey Optimizerの将来の展望

新しい Web チャネルを使用すると、マーケターは WYSIWYG のビジュアルデザイナーで Web キャンペーンを作成できます。 さらに、Journey Optimizerでは、E メール、Web、プッシュ、アプリ内、SMS など、任意のチャネルのコンテンツを試すことができるようになりました。
