# Web チャネルとコンテンツ実験を使用した Adobe Journey Optimizer の将来の展望

新しい web チャネルを使用すると、マーケターは WYSIWYG のビジュアルデザイナーで web キャンペーンを作成できます。さらに、Journey Optimizer では、メール、web、プッシュ、アプリ内、SMS など、任意のチャネルのコンテンツを実験できるようになりました。
