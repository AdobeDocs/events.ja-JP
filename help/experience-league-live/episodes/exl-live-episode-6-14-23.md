# Web チャネルとコンテンツ実験でAdobe Journey Optimizerの未来を探る

新しい web チャネルを使用すると、マーケターは WYSIWYG ビジュアルデザイナーで web キャンペーンを作成できます。 さらに、Journey Optimizerでは、メール、web、プッシュ、アプリ内、SMS など、任意のチャネルでコンテンツを実験できるようになりました。
