# 仕事のエージェント：オーディエンスとジャーニーの作成と活性化

現代のマーケティングは、プロンプトのスピードに合わせて動いています。 このライブウェビナーでは、Adobeのプロダクトチームが、Adobe Experience Platformがどのようにエージェンティック AIを日々のワークフローに導入するのかを紹介します。これにより、マーケターはより迅速にアイデアを立案し、コンテンツを制作して、活用できます。

Audience Agentで、オーディエンスのインサイトを明らかにし、すぐに使用できるオーディエンスを数分で作成できます。 さらに、Journey Agentが自然言語によるプロンプトから、オーディエンスの選定からクロスチャネルのアクティベーションに至るまでのマルチステップのジャーニーを実現する様子をご覧ください。  Real-Time CDPとJourney Optimizerに搭載されているエージェントが、業務ユーザーと技術ユーザーの両方に大規模な顧客体験を提供する際に、どのように手間を軽減するのかをご確認ください。
