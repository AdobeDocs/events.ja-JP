# 待っていてはいけません…共同作業！ Real-Time CDP Collaborationとは

アドビのエキスパートは、2025年4月10日に公開された当初、Real-Time CDP Collaborationがどのように企業やメディア企業にユーザーフレンドリーでプライバシー保護を重視したデータコラボレーションを提供するかを解説しました。 米国の顧客向けに提供されており、オーディエンスを発見、活性化、測定することで、有料マーケティングのペースに合わせて広告売上を最大化し、有料メディア施策を強化することができます。

このセッションでは、次のことについて解説しました。

* アドビが本製品を開発した理由と、それが消費者のプライバシー保護やオーディエンスへのリーチという課題を、シンプルさと相互運用性によって解決する方法をご紹介します
* Real-Time CDP Collaborationの活用事例（製品デモ付き）
* サポートされているユースケースとその展望
