# 顧客エンゲージメントの新しいユースケース：iOS ライブアクティビティ、web プッシュ、永続的なメッセージ

ブランドが一方向のシングルチャネルメッセージを超えて進化する中、顧客エンゲージメントの新しいサーフェスとユースケースを探ります。 マーケティングチームと製品チームが、より可視的でコンテキストに即した永続的なエクスペリエンスを構築し、継続的なエンゲージメントと、チャネルをまたいでより豊かでリアルタイムのインタラクションを促進する方法をご確認ください。

>[!VIDEO](https://video.tv.adobe.com/v/3483389/?quality=12&learn=on)

[Experience Leagueのコミュニティ投稿](https://experienceleaguecommunities.adobe.com/adobe-journey-optimizer-15/experience-league-live-new-use-cases-for-customer-engagement-ios-live-activities-web-push-and-persistent-messaging-249581){target="_blank"}で、番組に関するディスカッションを続けてください。

<br>
</br>
Adobe Adobe Journey Optimizerを利用している企業が、Adobe iOSのライブアクティビティ、web プッシュ通知、Message Inboxを活用して、カスタマーエンゲージメント戦略をどのように強化できるのかを解説しました。 これらの新しいチャネルと機能を組み合わせることで、リアルタイムの更新を実現し、モバイルアプリに留まることなくエンゲージメントを拡大し、重要なメッセージを常に利用可能にすることができ、一貫性のある顧客中心のジャーニーを構築できます。

対象は、次の通りです。

- iOSのライブアクティビティ：iPhoneのロック画面とDynamic Islandでリアルタイムに更新を提供して、注文追跡、フライト状況、ライブスコアなどの重要な瞬間を把握し、イベントの展開に合わせて顧客に情報を継続的に提供できます。

- web プッシュ通知：Journey Optimizerのモバイルプッシュ通知と同じユーザーワークフローを使用してweb サイトがアクティブに開いていない場合でも、web ブラウザー内で直接お客様にリーチすることで、モバイルアプリに留まらずエンゲージメントを促進します

- メッセージの受信トレイ：顧客が重要なメッセージ、パーソナライズされたコンテンツ、オファーを再訪問できる、永続的なブランドの受信トレイを作成することで、関連性とユーザーエクスペリエンスを向上させながら、通知の欠落を減らすことができます。
