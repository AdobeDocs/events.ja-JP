# 顧客エンゲージメントの新しいユースケース：iOS ライブアクティビティ、web プッシュ、永続的なメッセージ

ブランドが一方向のシングルチャネルメッセージを超えて進化する中、顧客エンゲージメントの新しいサーフェスとユースケースを探ります。 マーケティング部門と製品部門が、より可視的でコンテキストに即した永続的なエクスペリエンスを構築し、チャネルをまたいで継続的なエンゲージメントと豊かでリアルタイムのインタラクションを促進する方法をご確認ください。

Adobe Adobe Journey Optimizerを利用している企業が、iOS Live Activities、web プッシュ通知、Message Inboxを活用して、カスタマーエンゲージメント戦略をどのように強化できるのかをご確認ください。 これらの新しいチャネルと機能を組み合わせることで、リアルタイムの更新を実現し、モバイルアプリに留まることなくエンゲージメントを拡大し、重要なメッセージを常に利用可能にすることができ、一貫性のある顧客中心のジャーニーを構築できます。

主な内容：

- iOSのライブアクティビティ：iPhoneのロック画面とDynamic Islandでリアルタイムに更新を提供して、注文追跡、フライト状況、ライブスコアなどの重要な瞬間を把握し、イベントの展開に合わせて顧客に情報を継続的に提供できます。

- web プッシュ通知：Journey Optimizerのモバイルプッシュ通知と同じユーザーワークフローを使用してweb サイトがアクティブに開いていない場合でも、web ブラウザー内で直接お客様にリーチすることで、モバイルアプリに留まらずエンゲージメントを促進します

- メッセージの受信トレイ：顧客が重要なメッセージ、パーソナライズされたコンテンツ、オファーを再訪問できる、永続的なブランドの受信トレイを作成し、関連性とユーザーエクスペリエンスを向上させながら、通知の見落としを減らします。
