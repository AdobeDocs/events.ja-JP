# カスタマーエンゲージメントの新しいユースケース：iOS ライブアクティビティ、Web プッシュおよび永続メッセージ

ブランドが一方向のシングルチャネルメッセージを超えて移行する際の、顧客エンゲージメントに関する新しいサーフェスとユースケースを紹介します。 マーケティングチームと製品チームが、より可視化され、コンテキストに応じた、永続的なエクスペリエンスを作成することで、継続的なエンゲージメントを促進し、チャネルをまたいで豊富なリアルタイムインタラクションを実現する方法を説明します。

Adobe Journey Optimizerを使用する組織がiOS Live アクティビティ、web プッシュ通知、メッセージインボックスで顧客エンゲージメント戦略を拡張する方法について説明します。 これらの新しいチャネルと機能を組み合わせることで、チームは、リアルタイムのアップデートを配信し、モバイルアプリを超えてエンゲージメントを拡張し、重要なメッセージを常に利用可能にして、より一貫性のある顧客中心のジャーニーを作成することができます。

ここでは、以下について説明します。

- iOS ライブアクティビティ：注文の追跡、フライトステータス、ライブスコアなど、価値の高い瞬間に、iPhoneのロック画面と Dynamic Island でリアルタイムの更新を提供します。これにより、イベントが発生しても、お客様は継続的に情報を得ることができます。

- Web プッシュ通知：web ブラウザーで直接顧客にリーチし、Journey Optimizerのモバイルプッシュと同じユーザーワークフローを使用して web サイトがアクティブに開かれていない場合でも、訪問者を直接引き付けることで、モバイルアプリ以外にもエンゲージメントを広げます

- メッセージインボックス：顧客が重要なメッセージ、パーソナライズされたコンテンツおよびオファーに再度アクセスできる、永続的なブランドのインボックスを作成します。これにより、見逃した通知を減らしつつ、関連度とユーザーエクスペリエンスを向上させることができます。
