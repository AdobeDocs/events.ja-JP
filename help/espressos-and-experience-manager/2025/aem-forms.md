# エスプレッソス&amp;Experience Manager - AEM Forms

AEM Formsは、リードを獲得し、ロイヤル顧客化を促進するための重要な役割を担っています。 最適化されたフォーム入力体験、大規模なオーサリング、パーソナライズされたコミュニケーションにより、カスタマージャーニーのあらゆる接点でコンバージョン率を最大化します。 このウェビナーでは、Adobeのプロダクトチームが、AEM Formsのユースケースを紹介し、データのインプットとアウトプットに関する最も一般的な課題に対処します。 また、新しい生成AI アシスタント、AEM Sitesとの統合構成、会話を通じたフォームのデプロイ方法など、最新のイノベーションも取り上げています。 最後に、お客様の質問に対するQ&amp;A セッションをライブで行います。

>[!VIDEO](https://video.tv.adobe.com/v/3451636/?learn=on&enablevpops)

## 重要な留意点


* **パフォーマンス管理の課題**

   * 企業は、フォームの拡張、オムニチャネル体験の提供、テクノロジースタックの断片化など、さまざまな課題に直面しています。
   * 利用者は、迅速かつシームレスで統合された体験を期待しており、複雑性が発生した場合はフォームを放棄する3人中2人が利用しています。

* **AEM Forms ソリューション**

   * AEM Formsは、業務効率と優れたユーザーエクスペリエンスを提供することで、これらの課題に対処します。
   * 主な利点として、フォームの入力率、コンバージョン率、ユーザーエクスペリエンスの向上、顧客ミスの減少が挙げられます。

* **AEM Formsの機能**

   * オムニチャネルフォーム、自動化されたワークフロー、インタラクティブなカスタマーコミュニケーション、Adobe Experience Cloudとの統合など、エンドツーエンドのソリューションを提供します。
   * AIを活用し、パーソナライゼーションと効率性を強化。

* **ビジネスへの影響**

   * 例えば、オンライン登録のコンバージョン率が2.5倍、フォーム編集が85%高速化、顧客エラーが50%減少しました。

* **クラウドサービス**

   * クラウドネイティブのパフォーマンスにより、フォームを常に最新で拡張性が高く、安全で利用可能なものにします。
   * Edge delivery servicesは、待ち時間を短縮し、ユーザーエクスペリエンスを向上させます。

* **イノベーションとロードマップ**

   * 最近のイノベーションには、ドキュメントベースのオーサリング、対話型フォーム、ユニバーサルエディターのサポートなどがあります。
   * 今後の機能には、AI ベースのフォーム生成、マルチモーダルなフォーム入力、RDC ATP コネクタによるパーソナライゼーション、クラウドに移行するインタラクティブなコミュニケーションなどがあります。

* **AI統合**

   * AIを活用することで、チーム構造を置き換えることなく、フォームの作成、テストの自動化、効率性の向上を実現できます。

* **イノベーションへのアクセス**

   * 既存のクラウドをご利用のお客様は、Adobeにお問い合わせいただくか、提供されたフォームに入力して、早期イノベーションへのアクセスをリクエストできます。
