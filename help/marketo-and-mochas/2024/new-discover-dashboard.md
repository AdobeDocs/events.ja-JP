# 新しい「もっと知る」ダッシュボード

AdobeのシニアビジネスアドバイザーであるKate Colbert氏とテクニカルアドバイザーであるJames Leedom氏が、Marketo Measureのすべてのインスタンスで利用できる9つの新しい「もっと知る」ダッシュボードを詳しく説明します。 これらの新しくリリースされたダッシュボードは、収益、収益属性、ROI、エンゲージメント、リードベロシティ、商談ベロシティ、web トラフィックなどの重要な指標をまたいで、動的な視点と強化されたインサイトを提供します。 エンゲージメント戦略を洗練させ、成長とコンバージョンを加速させるには、データ主導のインサイトが不可欠です。

このセッションでは、これらの新しいダッシュボードをマーケティングレポートに最も効率的に組み込む方法を説明します。 また、Q&amp;Aの際も質問に答えます！

>[!VIDEO](https://video.tv.adobe.com/v/3428405/?learn=on)

## 重要な留意点

* Marketo Measureの「もっと知る」ダッシュボードは、使いやすさを向上させ、レポート機能に価値を加えるために設計された、事前構築済みのダッシュボード一式です。
* 古いダッシュボードは3月に廃止され、新しいダッシュボードはすべてのMarketo Measureのお客様が利用できるようになりました。
* 新しいダッシュボードには、データセグメンテーション、タッチポイント抑制、マーケティングパフォーマンスを分析および追跡するための様々な指標やビジュアライゼーションなどの機能が搭載されました。
* ボードは、売上の概要、売上への貢献度、ROI、リードパスポート、機会パスポート、ベロシティ、web トラフィック、エンゲージメントなど、さまざまな分野に広がっています。
* 特定のビジネスニーズとデータの正確性にもとづいてダッシュボードを設定することは、正確で有意義なインサイトを得るために不可欠です。
* 「もっと知る」ダッシュボードは、高度なトレンド分析とROIに適しています。一方、Salesforceレポートは、より詳細で、特定の個人やレコードに焦点を当てています。
* Data Warehouse オプションは、より高度なデータ分析機能とビジュアライゼーション機能を提供します。
