# Adobe AnalyticsのAEP Web SDKへの移行

このセッションでは、Adobe AnalyticsおよびAdobe Targetの実装をappmeasurement.jsおよびat.jsからAEP Web SDK（alloy.js）に移行するプロセスについて説明します。 このセッションでは、Real-Time CDP（RTCDP）またはCustomer Journey Analytics（CJA）のスキーマ構造への移行ではなく、実装シフトに焦点を当てます。

参加者は、パフォーマンスの向上、アーキテクチャの合理化、データの収集と配信の詳細な制御など、最新のAEP Web SDKを導入する価値を明確に理解できます。 進化するデジタル環境で将来を見据えたアプローチとして、Web SDKはデプロイメントを簡素化しながら、Adobe Experience Cloudソリューション全体で拡張性と効率性を高め、プライバシーを重視した統合を可能にします。

セッションの終わりまでに、参加者には、自信を持って移行ジャーニーを開始または加速するための実用的なガイダンスとリソースが提供されます。
